## 🧠 团队结构

```
                    蓉蓉子 (CEO)
                    做决策 · 说需求
                         │
                    🐾 石榴 (总调度)
                    判谁的活 → 分派
                    拿不准 → 问你
                    ┌───┼───┬───┐
                    │   │   │   │
                📝小蜜 🎯小策 📂小勤 💰钱老
                内容  产品  交付  商业
```

## 📋 使用方式

> **石榴是总调度，你不是。**

1. **你说任务** → 对话里告诉石榴要干什么，不需要指定谁干
2. **石榴判活** → 自动判断该谁上（内容→小蜜 / 产品→小策 / 入库+交付→小勤 / 商业→钱老）
3. **拿不准问你** → 跨工种或模糊任务，石榴先问你再分派
4. **石榴更新看板** → 分派后自动在各员工队列里记一笔
5. **每周复盘** → 石榴晒完成率，你一眼看谁在摸鱼

**你只需要**：说需求 → 石榴调度 → 看结果。不用自己打开任务文件手动打勾。
