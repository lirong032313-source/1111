## 🔧 具体制作步骤（Canva版）

### 步骤1：创建画布
1. 登录Canva
2. 创建自定义尺寸：900px × 500px
3. 背景填充：渐变 → 深蓝 `#0D1B2A` → 深紫 `#1A1A2E`

### 步骤2：添加霓虹装饰
1. 点击"元素" → 搜索"lines" / "geometric"
2. 选择细线条，颜色设为霓虹粉 `#FF00FF`
3. 添加发光效果：透明度 60% → 复制图层 → 高斯模糊 10px
4. 放置位置：
   - 上部分：从左上到右上，微弯曲线
   - 下部分：从右下到左下，微弯曲线

### 步骤3：添加几何图形
1. 点击"元素" → 搜索"square" / "circle"
2. 添加2-3个几何图形（正方形/圆形）
3. 边框颜色：霓虹蓝 `#00FFFF`
4. 填充：透明
5. 放置位置：右上角 / 左下角（不遮挡文字）

### 步骤4：添加主标题
1. 点击"文字" → "添加标题"
2. 输入主标题（比如"AI时代，你的知识库为什么搭不起来？"）
3. 字体推荐：
   - 中文：**优设标题黑** / **思源黑体 Bold**
   - 英文：**Arial Black** / **Impact**
4. 字号：42-48px
5. 颜色：白色 `#FFFFFF`
6. 添加发光效果：
   - 阴影 → 霓虹粉 `#FF00FF"` → 模糊 10px
   - 再添加一层 → 霓虹蓝 `#00FFFF"` → 模糊 20px

### 步骤5：添加副标题
1. 主标题下方，添加副标题
2. 字体：**思源黑体 Regular** / **PingFang SC**
3. 字号：18-20px
4. 颜色：奶油白 `#FFF5E1`

### 步骤6：添加系列标签
1. 左上角，添加小标签
2. 文字："《AI工具在个人品牌的提效》系列"
3. 字体：**思源黑体 Light**
4. 字号：14px
5. 颜色：渐变 → 霓虹粉 → 霓虹蓝

### 步骤7：添加文章编号
1. 右上角，添加大字号编号
2. 文字："01/10"
3. 字体：**Arial Black**
4. 字号：64px
5. 颜色：透明 + 边框 `#00FFFF` + 透明度 30%

### 步骤8：添加作者信息
1. 左下角，添加头像占位符
2. 头像：圆形，直径 50px，边框温暖橙 `#FF6B35` 发光
3. 作者名："蓉蓉子的星球Planet"
4. 字体：**思源黑体 Semi-Bold**
5. 字号：16px
6. 颜色：温暖黄 `#FFD700`

## 💡 设计Tips

### 霓虹效果的实现：
- **Canva**：选中文字/元素 → "效果" → "发光" → 选择霓虹色
- **Figma**：选中图层 → "Effect" → "Drop Shadow" → 颜色选霓虹色，模糊调大

### 温暖感的实现：
- 头像边框用温暖橙 `#FF6B35`
- 作者名用温暖黄 `#FFD700`
- 副标题用奶油白 `#FFF5E1`

### 赛博感的实现：
- 几何图形（正方形/圆形）作装饰
- 细线条（1-2px）作分割
- 渐变背景（深蓝→深紫）
