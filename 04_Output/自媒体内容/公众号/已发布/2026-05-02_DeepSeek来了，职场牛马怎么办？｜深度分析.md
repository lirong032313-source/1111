# DeepSeek来了，职场牛马怎么办？｜深度分析

这是蓉蓉子的星球Planet第65篇文章，希望你喜欢

有趣的灵魂终会相遇

  


之前写过一篇AI的崛起 [AI时代人类的未来在哪里？｜深度分析](https://mp.weixin.qq.com/s?__biz=Mzk0OTc2MDc5MQ==&mid=2247486139&idx=1&sn=c9cc0317cbd962a9e8cbaa40d0837084&scene=21#wechat_redirect)，蛇年开年DeepSeek爆火真实的让大家感受到，AI已来。

AI的速度之快，超出了很多人的预期。我们不仅能看到它在取代传统工作岗位方面的巨大潜力，还能发现它正在深入到人类生活的方方面面——从提供精准的情感疗愈建议，到做出比人类更加准确的算命预测，AI已经彻底改变了我们对工作的认知和对未来的想象。

然而，随着AI技术的不断进化，职场中的很多人，特别是那些依赖于重复性工作、低技能岗位的群体，开始感受到前所未有的压力。许多职场人不得不开始思考一个核心问题：**在AI能够代替我们的工作时，我们的真正竞争力究竟是什么？**

**01**

**AI快速崛起的背后**

AI的快速发展让我们感受到了技术革命的力量。从最初的自动化工具，到如今能够模拟人类情感、分析复杂数据并提供决策支持的AI系统，技术的进步令人瞠目结舌。以DeepSeek为代表的AI，不仅在提高工作效率、减少错误方面表现出色，它甚至能够预测市场趋势、提供个性化的情绪治疗建议，甚至为用户提供深度心理咨询，这些都让AI不仅仅局限于替代简单的工作任务，它已经开始进入到人类的内心世界。

这种技术的飞跃，意味着许多传统的工作岗位正面临着被取代的风险，尤其是那些高度依赖于重复性劳动的职位。职场中的“牛马”们——那些长期从事低附加值工作的人，如何面对这一变革，成为了当今社会亟待解决的问题。

**02**

**AI带来的职场冲击**

AI所带来的职场冲击，首先体现在**职业岗位的减少** 。过去，许多工作岗位的设立是基于重复性劳动和低技能要求，AI的迅猛发展让这些工作显得愈发不必要。比如传统的客服行业已经被AI所替代，许多公司已经开始使用AI智能机器人处理客户咨询，降低了对人工客服的需求。类似地，内容审核、数据录入员、行政助理等岗位也正面临被自动化工具取代的威胁。

其次，**技能的过时问题** 日益严重。随着AI的不断进化，过去那些依赖传统技能的工人，若没有进行技能更新和提升，必将遭遇职业瓶颈。例如，过去许多人通过人工处理数据或简单的文职工作谋生，但如今AI能够快速完成这些任务，且不知疲倦，这使得许多传统职能工作岗位逐渐消失。

**职业认同感** 的缺失也是一大问题。长期从事低技能岗位的劳动者，往往面临着工作的社会认同感较低和职业自豪感较弱。AI的崛起，更是加剧了这一问题——许多人发现自己在AI的高效处理下，变得不再重要。这种心理上的冲击，使得职场人士更为焦虑，尤其是在面对AI逐渐取代工作的现实时，许多人产生了深深的困惑：如果连工作都能被AI取代，那么自己在职场中的定位和价值又何在？

**03**

**核心竞争力的再思考**

AI已经展现了令人难以想象的强大能力，尤其在处理海量信息、快速做出决策、甚至与人类进行情感交流等方面。但尽管如此，AI仍然有局限性，**它无法像人类一样进行创造性思考、灵感迸发、解决前所未见的复杂问题** 。

因此，问题的核心回到了“我们是谁”这一问题：我们是AI的提问者，还是答案的提供者？

  * **提问者与创造者：**

在这个被AI主导的时代，最重要的不是谁能做得更快，而是谁能够提出更有深度的问题，谁能够进行创造性思考。在许多领域，AI仍然需要人类来为其提供明确的指引和方向——例如，在人工智能的开发、文化创意、科学研究等方面，AI依赖人类的创新和决策。因此，拥有提问和创造力的人，才是最具竞争力的群体。

  * **回答者：**

如果一个人只是依赖AI来回答问题，或者仅仅是依赖AI来执行操作，那么他将在未来的职场中处于被动地位。AI的出现，让很多传统行业中依赖重复性劳动的人，失去了原有的竞争优势。如果一个人仅仅是执行者，那么随着AI技术的发展，这类岗位将被逐渐淘汰。 

  





**04**

**职场“牛马”如何应对AI挑战？**

面对AI的冲击，职场中的“牛马”们应当如何应对，才能确保自己在新职场环境中的生存与发展？

  * **自我提升与技能更新：**

不断提升自己的技能，学习新兴技术，尤其是与AI相关的知识，将有助于适应新的工作要求。例如，掌握数据分析、编程技能、人工智能应用等，将使职场人士拥有更多的机会。同时，学习软技能，如沟通能力、团队合作能力、领导力等，也能增加个人在职场中的竞争力。

  * **转型与创新：**

如果传统的工作岗位和技能不再适应AI的时代，那么转型是必不可少的。职场“牛马”们可以思考如何将自己的经验与AI相结合，开创新的职业路径。例如，成为AI应用领域的专家，或者开发AI相关的产品和服务，利用AI提升自己的职业竞争力。

  * **专注于创造性与人性化工作：**

AI可以替代大量重复性、低附加值的工作，但它无法替代创造性工作和与人类情感互动的能力。因此，职场“牛马”们可以通过培养自己的创造力、艺术感知力、人际沟通能力等，进入那些需要高度创造性和情感交流的领域。例如，设计、营销、心理咨询等领域，仍然需要人类的智慧和感性判断

  * **找到内在热爱，开拓多管道收入：**

与其在工作中焦虑，不如主动探索自己，找到内心的热爱，把自己的潜能优势发挥到最大。而这也是人类独特的武器，即我们是宇宙独一无二的个体，我们可以成为自己想成为的人，AI永远取代不了的是人与人的链接和温度。未来人与人之间的深度互动会变得更加珍贵。尤其对于职场人来说，早日开启自己的职业第二曲线，开拓自己的多管道收入变得格外重要，主业副业并行不仅可以抵御职场中的风险，还可以帮助我们探索更多的可能性。

我们不必担心AI会不会取代人类，AI的到来只会让我们更加清晰的意识到作为人的独特性，重新回到“我是谁”的根本问题。

而职场中真正的核心竞争力，不在于是否能重复劳动，而是在于是否具备提出问题、创新思维的能力。我们每个人都应该意识到，AI的到来并不是要将我们淘汰，而是给了我们重新定义自己的机会——通过不断学习、创新与创造，去探索那些AI无法替代的领域，找到属于自己的职业新天地。
