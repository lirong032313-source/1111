# 一碗牛肉拉面的启示

**“** 毋以善小而不为，像做牛肉面一样做自己的MVP产品**”**  


  


  


这是蓉蓉子的星球Planet第11篇文章，希望你喜欢

有趣的灵魂终会相遇

  


  


今晚，我坐在北京跳海喝酒写一碗牛肉面的故事。

放以前，我可能没有这么敏锐的注意到身边的小事情，晚上在和面馆老板对话的过程中，突然冒出“要像做牛肉面一样做好产品”这样的念头，并且我发现真正的营销高手很多时候就在我们身边。  


  


先说下为什么很喜欢他们家的牛肉面，第一，面很劲道，是北方人喜欢的味道；第二，与别人店里服务不一样的是，那边的服务员和老板都很松弛，每次去都会笑着问“刚下班吗？今天想吃什么呀” ，今天如往常一样我说吃牛肉面，服务员说好，紧接着就说除了面她家还推了几款新的面和新品米饭套餐，看我有点犹豫，里面老板边拿着煮面的漏勺边说“我们推了一款臭鳜鱼米饭，这个鱼不好做，中午供应不过来，只有下午人少的时候开放，想让你们这些上班辛苦的人吃好一点”紧接着又说“不过没关系，你自己选择想吃的”

  


说到这里你们应该也猜到了，我点了他们家的新品臭鳜鱼米饭，点完后老板过来跟我说“你是这个新品的第一位顾客，我们拿不准大家喜不喜欢吃，你吃饭给我们提点建议”，说完又给我送了小菜，走的时候微笑的送走了我。

  


我的真实感受是，像我这样的顾客，重点不是在谁家吃面，而是更看重服务和提供的情绪价值，而今天，我花了38块体验了超值的牛肉面。在牛肉面这款产品背后还附加了情绪价值、真诚反馈和情感链接。尤其当他说推出鱼+米饭初心是想让辛苦的打工人吃好一点，牛肉面中加入这样简单的一句情感共鸣就足够让我这样的用户为产品买单了。

最近几天正在想产品的峰值体验，我们都知道像宜家、全季这样的连锁做的很成熟，从进店到过程体验再到离开，在用户的每一个关键旅程中都会设计考虑用户的峰值体验，让客户有“aha” moment 的惊喜。

  


然而，大部分普通人在做自己小而美的产品的时候，很难对标大厂的成熟的产品模式，从市场调研-竞品对标-产品设计-运营交付这样的流程冗长又费时，而很多人也是在做的过程中步步卡，卡久了就开始内耗或者干脆放弃。所以，我在吃完牛肉面后就在思考究竟什么是小而美的产品？小而美的产品应该具备哪些特征？普通人如果想做是否可行？  


  


  


01

—

什么是小而美的产品？

  


我理解的小而美的产品是在产品MVP中加入创意和灵魂，简单来说就是让ta变得有温度。同样是牛肉面，有温度的时候会让别人不仅会产品买单，还会人产生某种神奇的链接。

  


比如我们身边的社区咖啡、街角酒吧、还有楼下理发店等等，都是这样小而美的模式，甚至有些店带有极强的主人风格和私密性，去了ta家咖啡馆会有种进了ta家客厅的感觉，这是一种神奇的感受，与老板聊天的过程会有“朋友”的亲密感，明明你才去了没几次，如果你仔细观察，会发现这些店都很多共同的特性，并且他们都在践行“小而美”的创业模式。  


活得很好的小而美线下店：  


  1. 店面很小，一般在居民楼附近/写字楼附近；  


  2. 产品很单一，一进去就知道是什么店，总有一款是主打款；

  3. 老板很nice，有种不言而喻的亲切感，让你很想跟ta说话；

  4. 几乎不做营销，甚至你会怀疑ta会不会干倒闭




  


用产品的语言翻译下好的小而美产品模式：  


  1. 产品=功能价值+情绪价值，功能价值是固定的标准的，比如一杯咖啡，要有一款主打，给用户的情绪价值是无限的也是非标的。  


  2. 人群=特定人群，有共性的人群，比如都爱喝咖啡的人群；

  3. 服务=提供有温度的服务，比如店内的贴心布置标语、洗手间的贴图等等，还有老板的魅力温度让你想靠近；

  4. 营销=不销而销，好的销售是忘记成交和收费，更多的关心用户需要什么，持续产生深度链接。  


  





  


  


02

—  


小而美的产品应该具备什么？

  


首先，需要一个可行性的MVP产品，服务的人群足够聚焦，只要解决这些特定人群的特定问题就可以，重要是敏锐的觉察这些人群的需求和痛点，然后形成一个小而美的产品做服务。

  


比如我当初做快乐社群的时候发现很多职场人躺不平卷不赢，很想找到副业和职业第二曲线的突破口，那么我就结合链接职场人的同频社群+潜能优势测评+教练对话的方式帮助他们发掘潜能优势，解决情绪/生活卡点，更好的开启第二曲线；比如你发现身边有一群人睡眠有障碍，常常睡不好，可以用教练对话+精力管理+冥想辅导帮助他们解决睡觉问题。

  


第二，独特性和差异化。在提供功能价值中加入情绪价值和有温度的服务，这也是客户为什么买你的产品而不是别人的产品。  


  


最后，最重要的是人，有人就会有需求，有需求就会有产品。所以产品开始第一步，是去链接不同的人，也就是形成自己的场域，很重要。

  


  


03

—  


写在最后

  


我们的脑海中经常会冒出很多绝妙的idea，为什么没有变成现实？甚至还没开始就感到焦虑？

  


是因为你内心不想做，不想做的背后可能是害怕失败或者听到他人的负反馈，可以问自己如果你做了会发生什么？那些他人的评判是真实的吗？如果一直拖延，会发生什么？

  


当你想做的时候只要行动一小步就好，我们总是想的太多做的太少，其实只要每天行动一小步，像做牛肉拉面一样做自己的MVP产品就好。
