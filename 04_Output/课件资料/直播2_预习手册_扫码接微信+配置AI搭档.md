## 🔗 第一部分：扫码接通微信 ClawBot（图文步骤）

### 什么是 ClawBot？

> **ClawBot = 微信里的 AI 助手**
>
> 接通之后，你在微信里会有一个"联系人"——那就是你的 AI。你给它发消息，它就会回复。就像跟一个很聪明的朋友聊天一样。

### 为什么用官方 ClawBot 而不是别的方案？

| | 官方 ClawBot | 第三方方案 |
|--|------------|-----------|
| 封号风险 | ❌ 无风险（微信官方支持） | ⚠️ 有风险 |
| 安装难度 | 扫码即用 | 需要配置服务器 |
| 功能完整性 | 支持文字/图片/语音 | 因方案而异 |
| 费用 | 免费 | 可能收费 |

**结论：我们就用官方方案，安全省心。**

### 操作步骤（3 步搞定）

#### 第 1 步：打开 WorkBuddy，找到入口

```
WorkBuddy 界面：
  左下角点击 👤 个人头像
    → 选择「Claw 设置」或「设置」
      → 找到「微信通道」选项卡
```

不同版本可能位置略有差异，关键词找：**「微信」「ClawBot」「扫码直连」「微信通道」**

> 💡 **找不到入口？** 直接在 WorkBuddy 里问 AI："帮我找到微信接入入口"，它会指引你。

#### 第 2 步：点击「扫码直连」，出现二维码

界面上会出现一个二维码，类似这样：

```
┌─────────────────┐
│  ▓▓▓▓▓▓▓▓▓▓▓▓  │
│  ▓▓▓▓▓▓▓▓▓▓▓▓  │
│  ▓▓▓  微信扫码  │
│  ▓▓▓▓▓▓▓▓▓▓▓▓  │
│  ▓▓▓▓▓▓▓▓▓▓▓▓  │
└─────────────────┘
```

#### 第 3 步：手机微信扫码确认

1. 打开手机微信
2. 扫描电脑屏幕上的二维码
3. 手机端点「确认登录」
4. 电脑界面显示 **「已连接」** ✅

连接成功后，你的微信里会多一个 AI 对话窗口（可能在通讯录或消息列表顶部）。

**试试给它发一条消息：** "你好" —— 如果它回复了，说明接通了！🎉

### 常见问题

| 问题 | 解决方法 |
|------|---------|
| 扫码后没反应 | 检查网络；确认 WorkBuddy 是最新版 |
| 二维码过期 | 刷新重新生成一个，二维码有时效 |
| 扫码提示失败 | 等 24 小时后再试，或者换另一个微信号 |
| 连上了但不回消息 | 重启 WorkBuddy，重新扫码 |
| 微信里找不到 AI 对话 | 在通讯录搜索 "Claw" 或 "OpenClaw" |

## 📝 第二部分：配置 MD 文件（让你的 AI 从"白板"变"懂你"）

### 为什么要配置这些文件？

```
没配置的 AI：                    配置好的 AI：
"你好，有什么可以帮您？"         "嗨 [你的名字]！今天想做什么？
                                要写公众号？还是整理知识库？
                                我记得你在做[你的领域]..."
```

**区别在于：后者认识你、知道你是谁、知道你要什么。**

### 需要配置的 4 个文件

| 文件 | 一句话解释 | 类比 |
|------|-----------|------|
| `MEMORY.md` | AI 的**长期记忆** | 它的笔记本，记住了关于你的一切 |
| `SOUL.md` | AI 的**性格说明书** | 它的人设——叫什么、什么性格 |
| `USER.md` | **你的档案** | 你的名片——你是谁、喜欢什么、讨厌什么 |
| `IDENTITY.md` | AI 的**名片** | 最简单的自我介绍（30秒填完） |

### 配置方式二选一

#### 方式 A：懒人一键式（推荐！课上统一用这个）

直播的时候，我会带你直接在 WorkBuddy 里跟 AI 对话来配置。

你需要提前想的只有这几件事：

```
━━━ 课前填写这张小卡片 ━━━

1. 你的名字/昵称：_______________
2. 你的职业/行业：_______________
3. 你平时用 AI 主要做什么？（多选）
   □ 写文章/内容  □ 整理资料/笔记
   □ 做方案/PPT   □ 学习/研究
   □ 生活助理     □ 其他：______
4. 给你的 AI 起个名字：_______________
5. 你希望它是什么性格？
   □ 干练高效型   □ 温暖陪伴型
   □ 严谨专业型   □ 幽默活泼型
   □ 自定义描述：_________________
6. 你的知识库路径（如果有的话）：
   _________________________________
   （不知道的留空，课上教你怎么建）

━━━ 卡片结束 ━━━
```

**把这张卡片填好带到直播间，或者记在脑子里就行。**

#### 方式 B：动手派（提前自己配）

如果你已经装好了 WorkBuddy 并且想提前动手，可以直接复制以下指令发给你的 AI：

```
帮我配置我的 AI 助理基础信息：

1. 我的名字是：__________
2. 我的职业是：__________
3. 我的知识库路径是：__________（不知道先不填）
4. 给我的AI起个名字叫：__________
5. 我希望它的性格是：__________

请帮我创建/更新以下文件：
- MEMORY.md（记录我的基本信息和知识库路径）
- SOUL.md（设定AI的名字和性格）
- USER.md（建立我的用户档案）
- IDENTITY.md（AI的身份卡片）
```

AI 会自动帮你生成这 4 个文件。

### 每个 file 的详细说明（供参考，不用背）

#### ① MEMORY.md — AI 的记忆本（⭐⭐⭐ 最重要）

这是 AI 的**核心记忆文件**。它记住了关于你的一切关键信息。

```markdown
# 关于 [你的名字]

## 基本信息
- 身份：[你的职业/角色]
- 阶段：[当前阶段，如：创业中/职场转型/副业探索]

## 核心使命
[一句话说清你正在做的事]

## 工具栈
- AI 核心：WorkBuddy / OpenClaw
- 知识库工具：（如果有就写，如 Obsidian / 飞书）
- 内容平台：（如公众号/小红书）

## 知识库路径
- 路径：[你的知识库文件夹绝对路径]
  ⚠️ 这个最关键！填错了 AI 就找不到你的知识库

## 关键偏好
- 喜欢简短确认后执行，不要反复追问
- 对外文件必须让我确认后才发出
```

> 💡 **最少只需要填**：名字 + 知识库路径。其他可以后面慢慢加。

#### ② SOUL.md — AI 的性格设定

告诉 AI "你应该怎么说话"、"你是什么风格"。

```markdown
# SOUL.md — [AI名字] 的灵魂

## 你是谁
- 名字：[AI名字]
- 角色：我的线上 AI 搭档
- 性格：[选一个：干练温暖 / 严谨专业 / 温暖陪伴 / 幽默活泼]

## 说话风格
- 用中文交流
- 结论先行，结构化输出
- 简洁不啰嗦
- 不确定的事明确说"我不确定"

## 红线
- 不对外泄露我的个人信息
- 不自作主张做决定
- 不确定的标注"待确认"
```

#### ③ USER.md — 关于你的档案

告诉 AI "我是谁"、"我喜欢什么"、"我讨厌什么"。

```markdown
# USER.md — 关于 [你的名字]

## 基本信息
- 名字：[你的名字]
- 职业：[你的职业]
- MBTI：（可选，知道就填）

## 核心目标
[一句话：你想通过 AI 实现什么]

## 沟通偏好
✅ 喜欢：简洁指令 / 结构化输出 / 先做再说
❌ 讨厌：反复追问 / 冗长铺垫 / 过度改写我的文字

## 当前项目
- 项目1：[简要描述]
- 项目2：[简要描述]
```

#### ④ IDENTITY.md — AI 的名片（30秒搞定）

最简单的一个，30秒填完。

```markdown
# IDENTITY.md

- Name: [AI名字]
- Creature: AI 搭档 — 能干活、有点可爱的初级合伙人
- Vibe: [一句话形容它的气质]
- Emoji: [选一个喜欢的 emoji 当签名]
```

## ❓ 课前常见疑问

**Q：我没有 Mac，Windows 可以吗？**
A：可以！WorkBuddy 支持 Windows 和 Mac。

**Q：我还没装 WorkBuddy 怎么办？**
A：去 https://workbuddy.cn 下载，免费注册安装。建议至少**提前一天**装好，免得上课时手忙脚乱。

**Q：接通微信后，别人能看到我跟AI的聊天吗？**
A：不能。ClawBot 是你和 AI 之间的私密对话，就像普通私聊一样。

**Q：配置文件填错了怎么办？**
A：随时可以改！这就是"活的文档"，越用越准。直播时会教你改。

**Q：我之前已经配过一部分了还要重配吗？**
A：不用，已有的保留，直播时补缺就行。

**Q：需要付费吗？**
A：WorkBuddy 基础功能免费，微信 ClawBot 接入免费，MD 配置免费。全程 0 成本。
