# 搭子互审四维表

> 用于 Day 18 内容发布前搭子互审

| 维度           | 评分 (1-5) | 问题            |
| ------------ | -------- | ------------- |
| **1. 内容有价值** | ⭐⭐⭐⭐⭐    | 对读者有用吗？有干货吗？  |
| **2. 语言像你**  | ⭐⭐⭐⭐⭐    | 像你说话吗？有你的味道吗？ |
| **3. 结构清晰**  | ⭐⭐⭐⭐⭐    | 读得下去吗？逻辑顺吗？   |
| **4. 有行动点**  | ⭐⭐⭐⭐⭐    | 读者知道该做什么吗？    |

---

## 评分说明

| 分数 | 含义 |
|---|---|
| ⭐⭐⭐⭐⭐ 5分 | 完美，基本不用改 |
| ⭐⭐⭐⭐ 4分 | 很好，小改即可 |
| ⭐⭐⭐ 3分 | 一般，需要调整 |
| ⭐⭐ 2分 | 不太好，要大改 |
| ⭐ 1分 | 不行，得重写 |

1. 搭子双方互换文章
2. 各自用四维表打分
3. 写下具体反馈
