## 📦 结营附赠：CLAUDE 更新 + Lint 体检

### 📝 CLAUDE 更新规则（日常对话完成）

**场景**：当 AI 写的内容"不像你"时

**触发**：直接告诉 AI，不需要单独 Skill

**操作步骤**：
```
1. 指出："这段写得不像我"
2. 说明："太正式/太营销/太AI..."
3. AI 自动更新 CLAUDE.md
4. 验证：再写一次
```
