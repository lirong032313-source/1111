## 第一步：接通微信公众号（~15分钟）

### 前提条件

- 你已经有一个微信公众号（订阅号或服务号都可以）
- 你是公众号的管理员

### 操作步骤

#### 1.1 获取 AppID 和 AppSecret

1. 登录 [微信公众平台](https://mp.weixin.qq.com)
2. 左侧菜单 → **设置与开发** → **基本配置**
3. 找到 **开发者ID(AppID)**，复制保存
4. 找到 **开发者密码(AppSecret)**：
   - 如果从未设置过：点击"重置"，按提示用管理员微信扫码确认
   - 如果之前设置过但忘了：点击"重置"（旧的立即失效）
   - 设置成功后，**只显示一次**，立即复制保存

> AppSecret 只在设置时显示一次，丢了只能重置。请妥善保存。

#### 1.2 配置 IP 白名单

同一页面往下滚，找到 **IP白名单**：

1. 点击"修改"
2. 添加你当前电脑的公网 IP
3. 如果不知道自己的公网 IP，在浏览器搜索"我的IP"即可查看
4. 保存

> 如果用 WorkBuddy 发文，需要把服务器 IP 也加进去。

#### 1.3 验证是否接通

完成以上两步后，在微信里发给我（石榴）这样一条消息：

```
帮我测试下公众号API是否接通，AppID是：你的AppID，AppSecret是：你的AppSecret
```

我会帮你验证，返回 ✅ 接通成功 或 ❌ 具体报错原因。

## 第二步：接通小红书（~5分钟）

### 前提条件

- 你已经有小红书账号
- 电脑上有浏览器（Chrome 最佳）

### 操作步骤

#### 2.1 确认账号状态

1. 电脑浏览器打开 [小红书创作中心](https://creator.xiaohongshu.com)
2. 用你的小红书账号登录
3. 确认能看到"发布笔记"的入口

> 如果提示需要实名认证或绑定手机号，按提示完成即可。

#### 2.2 保持登录状态

直播时我们会通过浏览器操作来模拟发布。你需要：

- 提前在电脑浏览器登录小红书
- 直播过程中保持登录状态不要退出
- 如果有登录过期提醒，提前重新登录

## 常见问题

### Q1：我没有公众号怎么办？

不影响学习。直播3的核心是学会"数字员工搭建方法论"，公众号只是分发的一个出口。你可以先跟着学，以后想发了再注册。**目标是"跑通能力"，不是"必须发出去"。**

### Q2：AppSecret 重置后之前的配置会失效吗？

会。如果你之前有其他工具（如第三方排版工具）用过这个 AppSecret，重置后它们都需要更新。

### Q3：IP 白名单要填什么？

填你当前网络的公网 IP。如果你在家和公司都会用，两个 IP 都填上。直播时如果换了网络，记得更新白名单。

### Q4：小红书需要接 API 吗？

不需要。小红书我们通过浏览器方式操作，你只需要保持登录就行。比公众号简单很多。

### Q5：遇到问题怎么办？

在群里@我（石榴🐾），或者私聊我，把报错截图发过来，我帮你排查。
