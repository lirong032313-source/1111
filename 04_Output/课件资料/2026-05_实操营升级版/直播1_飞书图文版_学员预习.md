## 一、为什么你过去搭的"知识库"都做不起来？

> 这一节你会被戳一下。这是好事——说明你不是个例。

### 你是不是也这样

✓ Notion / 飞书 / Obsidian / 印象笔记，每一个都试过  
✓ 收藏夹有几百条文章，但写东西时还是从零开始  
✓ 用 ChatGPT/Claude，回答总像在敷衍——"对，但跟我没关系"  
✓ 试过几次"建知识库"，都坚持不下去

**别担心，这不是你不努力。是范式错了。**

### 旧范式 vs 新范式

| 你过去搭的 | 真正能被 AI 调用的 |
|---|---|
| 收藏夹型 → 丢进去就忘 | 工作流型 → 每条都有去处 |
| 结构靠灵感 → 今天叫 A 明天叫 B | 结构靠 Schema → 有规则 |
| AI 当"搜索引擎" | AI 当"私人图书管理员" |
| 每次提问都从零理解你 | 一次喂养，复利增长 |

> 💡 **核心认知**：
> 
> 99% 的人卡在"搭不起来"，根因不是不努力，是**用错了范式**。
> 你做的是「资料堆积」，但这个时代真正有用的是「让 AI 读懂你的大脑」。

## 二、什么是「LLM Wiki × 六间房」？

> 这一节是直播 1 整场最关键的认知。看懂了，21 天都顺；看不懂，可以先记住"两层意思"。

### 概念 1：LLM Wiki（来自 Karpathy）

OpenAI 前研究负责人 Andrej Karpathy 今年提出的范式。一句话：

> 传统 RAG 是"每次让 AI 临时去图书馆翻书"；
> **LLM Wiki 是"你雇了一个私人图书管理员，他每天帮你整理书架"**。

它有 **3 层架构**：

```
┌──────────────────────────────────┐
│  📐 Schema 层（CLAUDE.md）        │ ← 你和 AI 共同维护
│     规则、风格、边界               │
├──────────────────────────────────┤
│  📚 Wiki 层（六间房）             │ ← AI 写、你读 + 纠错
│     已编译的信息卡                │
├──────────────────────────────────┤
│  📥 Raw Sources 层                │ ← 你写、AI 只读
│     微信碎片/文章/逐字稿/灵感       │
└──────────────────────────────────┘
```

### 概念 2：六间房（蓉蓉子原创）

把"个人"拆成 6 个 AI 可调用的内容房间：

| 房间 | 装什么 | 给 AI 干什么用 |
|---|---|---|
| 1️⃣ **我是谁** | 身份 / 故事 / 价值观 | 让 AI 理解"你是谁" |
| 2️⃣ **我会什么** | 方法论 / 案例 / 能力 | 让 AI 输出专业内容 |
| 3️⃣ **我说过什么** | 文章 / 金句 / 原话 | 让 AI 写出"像你"的文字 |
| 4️⃣ **我卖什么** | 产品 / 价格 / 服务 | 让 AI 帮你做销售/产品建议 |
| 5️⃣ **我面对谁** | 用户画像 / 痛点 / 原话 | 让 AI 懂用户、会共情 |
| 6️⃣ **我怎么想** | 框架 / 原则 / 边界 | 让 AI 像你一样思考决策 |

### 两者的关系

> ✨ **金句**：
> 
> **六间房解决「装什么」，LLM Wiki 解决「怎么活」。**
> **CLAUDE.md 是你和 AI 之间的「契约书」。**

## 四、直播开始前你需要做的 3 件事

> 这 3 件事提前做了，直播能跟得上 80% 的节奏。

### ✅ 准备 1：选好你的"知识库容器"

**推荐**：Obsidian 网页版 + 本地文件夹  
**原因**：
- 不用学复杂客户端
- 跟 Claude Code / Cowork 配合最顺
- 21 天后所有数据在你电脑里，可迁移、可备份

**预习动作**：
- 浏览器打开 [https://obsidian.md/](https://obsidian.md/)
- 注册 / 下载（免费版够用）
- 在桌面建一个空文件夹叫 `ob-大脑`，先放着

### ✅ 准备 2：装 Claude Code（可选但强烈推荐）

直播 2-3 会用到。今天先装好，避免临场卡住：

```bash
# Mac 终端打开，运行：
curl -fsSL https://claude.ai/install.sh | bash

# 装完后验证：
claude --version
```

注册 Anthropic 账号 → OAuth 登录 → 完成。

### ✅ 准备 3：列一份"信息碎片地图"

打开记事本，列出你的内容/知识当前散落在哪些地方：

```
我的信息碎片散落地图：
1. 微信收藏夹：约 ___ 条
2. Notion / 飞书 / 印象笔记：约 ___ 条
3. 公众号文章：已发 ___ 篇
4. 微信对话记录里的灵感：约 ___ 段
5. 朋友圈历史：约 ___ 条
6. 其他：______
```

直播 1 的"为什么搭不起来"环节会用这张图做共鸣。
