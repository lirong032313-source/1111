## 📊 问卷使用建议（蓉蓉子参考）

### 1️⃣ 用于"搭子配对"（基于 Q11）

```
技术维度评分 = Q11b 选项数（0-5）+ Q7 含 Cursor/Copilot 加 1 分
内容维度评分 = Q4（阶段 C/D 加 2 分，B 加 1 分）
偏好补充 = Q11a + Q11c

匹配规则：
  ─ 技术高 + 内容高 → 自由匹配
  ─ 技术高 + 内容低 → 配 内容高 + 技术低
  ─ 技术低 + 内容低 → "双弱组"，单独配 mentor
  ─ Q11c 时段重合 ≥2 段 → 优先匹配
```

### 2️⃣ 用于"直播节奏调整"

| 数据来源 | 影响哪场直播 |
|---|---|
| Q6 频率 + Q7 工具栈 | 直播 1 - 工具讲解节奏；ChatGPT 用户多 → 多对比 Claude |
| Q5 知识管理状态 | 直播 1 - 三层架构讲解的细致度（有 Obsidian 经验的快讲；纯小白慢讲） |
| Q8 卡点（开放题） | 直播 1/2/3 都用，**Q8 文本是直播彩蛋金句的来源** |
| Q9 多选分布 | 决定 3 次直播的"主线"侧重 |
| Q10 文本 | 直播 1 开场可以引用 2-3 条："我们今天就要做这件事" |

### 3️⃣ 用于"开营前 1v1 私聊"（沿用 3月版做法）

把 Q8 + Q10 文本逐条看完，让 AI（按 3月版"AI 分身 4：作业批改官"模式）生成"个性化路径建议"初稿，你做 30% 人味校对，私聊每位学员：
> "看到你说希望 21 天后能 [____]，我帮你拆解了 3 步路径..."

> 💡 这是 33 人那期"学员被看见"的关键动作，5月版必须复用。

### 4️⃣ 用于"开营仪式感"（基于 Q12 行动承诺）

3月版数据显示：**完成全部承诺勾选 + 签名的学员，最终完课率显著高于只勾部分的**。

5月版可以做的强化：
- 把每位学员的"签名 + 行动承诺"做成开营当天发到社群的「承诺墙」拼图
- 21 天后结营时回放：你 21 天前承诺了什么 vs 你做到了什么
