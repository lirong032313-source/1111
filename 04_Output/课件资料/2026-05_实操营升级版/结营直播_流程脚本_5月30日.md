# 🎬 5月实操营 · 结营直播脚本

> **时间**：5月30日（周五）晚 20:00-21:30
> **时长**：~90分钟
> **平台**：腾讯会议 + 社群同步
> **目标**：成果展示 → 颁奖激励 → 升级转化

---

## 环节01 · 开场 · 三场直播串主线（10min）

### 核心思路
**不依赖打卡数据统计（可能无法精确统计）**，改用三场直播的内容回顾来串联21天的成长叙事线。每场直播代表一个成长阶段，形成"从0到1"的故事弧线。

### 操作清单
- [ ] 准备3场直播的核心金句/截图（各1-2张）
- [ ] 准备3-5张学员高光时刻图片/截图轮播
- [ ] 背景音乐：轻快但不抢戏的BGM

### 开场话术框架

```
"大家好！欢迎来到5月实操营的结营仪式。

21天前，我们在这里说——
'21天后你拥有的不是收藏夹，而是一只会自己长大的AI助手。'

这21天，我们一起走了这样一条路：

【直播① · 地基搭建】
那天我们说——
'AI不是替你干活，AI要先读懂你。'
你们学会了写信息卡，搭建了第一间房。
很多人第一次意识到：原来我的经验是可以被'装起来'的。

【直播② · 养龙虾】
第二次直播，我们把龙虾养起来了——
CLAUDE.md就是你的龙虾的'喂养手册'。
你开始发现：同样的提示词，有了知识库的AI和没有的是天壤之别。

【直播③ · 数字员工】
第三次直播，我们把AI从'工具'变成了'员工'——
Wiki × Skills × Workflow = 你的数字团队。
有人已经造出了自己的第一个数字员工。

（快速闪回3-5张高光截图：第一次成功运行 / 第一篇产出 / 群里互助的瞬间）

21天，从'AI是什么'到'AI怎么为我工作'。
这不是三门课，这是一条完整的路。

这些不是AI替你们做的。是你们自己做的。
我只是给了你们一把铲子，是你们自己挖出了宝藏。

接下来，我要请几位同学上台，
亲自给大家看看——这21天，他们挖到了什么。"
```

### 💡 执行Tips
- **叙事弧线是关键**：直播①②③分别对应 懂我→养活→替我工作，层层递进
- 高光闪回选**有代表性的瞬间**：第一次成功运行、第一次产出、群里一次感人的互助对话
- 控制在10min内，不拖沓，把时间留给学员展示

## 环节02 · 学员成果展示 · 核心高光（35min）

### 学员邀约（提前3天完成）

选4-5位有代表性、表达能力OK、成果可展示的学员：

| 类型 | 展示内容 | 邀约话术 |
|---|---|---|
| **成长型** | 从完全不会到跑通全流程 | "你的前后对比很有感染力，愿意分享下吗？" |
| **深度型** | 六间房搭建得最完整 | "你的知识库结构特别清晰，想请你做示范" |
| **产出型** | 已有文章/Skill/工作流成品 | "你已经跑通了，是最好的活广告" |
| **互助型** | 搭子互动多、帮助他人 | "大家对你的评价很高，来分享下心得吧？" |

### 每人分享框架（5-6min/人）

```
【固定格式 — 提前发给他们】

1. 来的时候我是什么状态？（1-2句）
2. 这21天我最大的1个改变是什么？（核心亮点）
3. 屏幕共享：展示你的成果（2-3min）
   · CLAUDE.md 或等价的配置文件
   · 六间房的结构（哪怕只搭了2-3间也行）
   · 第一篇Output（文章/卡片/任何产物）
4. 一句话送给还没开始/正在犹豫的人
```

### 蓉蓉子点评模板（每人1min）

```
"谢谢XX的分享。我注意到一个细节——
（提炼1个具体的观察点，证明你认真听了）

这恰恰说明了一个道理：
（升华到方法论层面，金句收尾）

给XX鼓掌👏"
```

### ⚠️ 关键执行点
- **提前彩排**：至少跟每位学员过一遍内容和屏幕共享操作
- **控制节奏**：每人严格5-6min，超时温柔打断："太精彩了，我们待会私下再聊"
- **制造反差**：开场说"来的时候"时强调痛点，展示时强调变化，对比越强烈越打动人
- **录屏留存**：学员同意的话，这些展示素材后续可以做销售案例

## 环节03+04 · 颁奖仪式（15min）

### 🥇 最佳搭子奖（TOP3，~7min）

#### 评选核心原则
搭子奖评的是 **"1对1双向陪伴的质量"** —— 不是谁最优秀，而是谁最懂得和另一个人一起变好。

#### 可量化评选标准（满分100分）

| 维度 | 权重 | 数据来源 | 计分方式 |
|---|---|---|---|
| **互评完成度** | **40%** | 互评表/问卷星收集 | 每次按时提交+有实质内容得满 分，每缺一次扣5分，**满分40分** |
| **搭子互助深度** | **35%** | 互评表中"对方帮助了我什么" | 搭子打分（0-35分）+ 蓉蓉子抽检验证，**满分35分** |
| **自身成长可见度** | **25%** | 作业抽查 + 结营展示 | 自己的作业/产出有明显进步、能展示成果，**满分25分** |

> 💡 **省力做法（强烈推荐）**：结营前3天发《搭子互评表》（问卷星），3个问题：
> 1. 你的搭子帮你最多的一件事是什么？
> 2. 你给搭子的帮助中，你觉得最有价值的一次是什么？
> 3. 如果满分10分，你给这位搭子打几分？
>
> 这比你自己翻21天聊天记录靠谱100倍。你只需要做最终裁定。

#### 颁奖流程

```
1. 蓉蓉子宣布："接下来是我们实操营的重头戏——最佳搭子奖"
2. 公布评选标准（让所有人知道规则透明）
3. 从第三名开始揭晓（制造悬念，压轴放第一名）
4. 每位获奖者：
   - 上台（开摄像头）
   - 蓉蓉子念颁奖词（1段 personalized 的认可）
   - 宣布权益：私教抵扣¥2000 + 实操营学费全退
   - 获奖者感言（1-2句即可，不要太长）
5. 合影留念（截图）
```

#### 📝 颁奖词模板（需要根据实际学员定制）

```
"第X名 —— @XXX！

这21天里，你做了XX件事让我印象特别深刻：
（具体事例，比如：每天最早打卡 / 帮搭子改了XX次 / 在群里分享了XX方法）

你不是一个人在学，你是带着别人一起走。
这就是'搭子'的意义。

🥇 最佳搭子权益：
· 私教升级抵扣 ¥2,000
· 实操营学费 全额退还
恭喜你！"
```

### ⭐ 互助之星奖（TOP5，~8min）

#### 评选核心原则
互助之星评的是 **"1对多的社群贡献度"** —— 与搭子奖的区别：搭子是深度1对1，互助星是广度1对多。

#### 可量化评选标准（满分100分）

| 维度 | 权重 | 数据来源 | 计分方式 |
|---|---|---|---|
| **社群活跃质量** | **40%** | 蓉蓉子/助教快速过群记录 | 有效发言数（排除"收到""好的"） + 主动分享经验次数，**满分40分** |
| **主动帮助他人** | **35%** | 群记录关键词搜索 | 每帮助1人得5分（搜"帮你看了""试试""可以这样"），上限**35分** |
| **正能量氛围贡献** | **25%** | 群记录 + 学员提名 | 鼓励他人/活跃气氛的次数，**满分25分** |

> 💡 **省力做法（强烈推荐）**：结营前2天发一张简单问卷（3个问题）：
> 1. 这21天谁在群里帮到你最多？（可多选）
> 2. 谁的发言让你印象最深刻？
> 3. 如果你要提名一位"互助之星"，你选谁？
>
> 让学员互相提名+自荐。你只需要做最终裁定。

#### 颁奖流程（同上，略）

#### 权益宣布模板

```
⭐ 互助之星权益：
· 私教升级抵扣 ¥1,000
· 实操营学费 全额退还
· 赠送 30分钟 1v1 诊断咨询
```

### 💡 颁奖环节设计巧思

1. **悬念设计**：从后往前公布名次，第一名最后出场
2. **视觉冲击**：准备简单的奖状图/PPT页面，宣布时切换过去
3. **社交证明**：获奖者的故事本身就是最好的转化素材
4. **不强迫表态**：获奖后问"你对下一步有什么想法？"——自然引出升级话题，但不当众逼单

### 🔄 群保留 vs 虾玩星球 · 差异化策略

**为什么群不解散但不影响虾玩星球转化？**

| 维度 | 实操营群（结营后） | 虾玩星球 |
|---|---|---|
| **定位** | 校友录（老同学偶尔聚聚） | 活的圈子（持续成长） |
| **运营频率** | 偶尔（蓉蓉子一周≤1条链接） | 高频（每月4场活动） |
| **核心价值** | 找到老同学、自由交流 | 直播/工作坊/吐槽大会/1v1诊断 |
| **AI能力** | 无 | 虾球助手24h在线 |
| **价格** | 免费 | ¥1,999/年 |

**关键话术（融入收尾环节）：**
> "群会留着做校友录，大家随时能找到老同学。但如果想要持续的、有人带着往前走的感觉——那是虾玩星球的事。"

**差异化本质**：免费的是"能找到人"，付费的是"有人带着走"。

## 环节05 · 结营升级路径指引（20min）⭐ 转化关键

### 第一步：两条路的选择（5min）

```
"21天结束了。但我知道，在座的很多人——
不是觉得'终于结束了'，
而是觉得'才刚刚开始'。

如果你觉得自己在AI这条路上想继续深耕，我有两个地方给你：

第一条路：🦐 虾玩星球
这不是课程，也不是训练营。是一个圈子。
每月4场活动，虾球助手24小时在线，
每年还有1次1v1深度诊断。
适合想持续迭代、但暂时不需要深度陪跑的你。

第二条路：💎 私教团
如果你已经尝到了甜头，
而且你有自己的产品或内容基础，
想要更系统地把它变成一套能赚钱的东西——
私教团就是为你准备的。"
```

### 第二步：私教门槛说明（5min）

```
但私教团有一个前提门槛——
不是钱的问题，是'有没有东西可以加工'的问题。

我说得很直白：
私教是用来帮你把'已有的'变得更值钱的。
如果你连'已有的'都没有，
那我给你的所有方法都落不了地。

所以私教团的门槛是：
✅ 有产品（课程/咨询/服务/内容，什么都行）
✅ 有内容（文章/案例/经验沉淀，哪怕是雏形）
或者 ✅ 已经购买了我的1v1知识库重构咨询

这三条满足任意一条，我们就可以聊。"
```

### 第三步：权益透明化（5min）

> ⚠️ 此处建议用PPT/共享屏幕展示表格

```
"很多同学关心的是——如果我要升级，到底要花多少钱？

我把账算得清清楚楚，不留任何模糊空间：

（展示权益表）

| 你的身份              | 私教原价    | 抵扣后实付约 |
|---------------------|----------|----------|
| 🥇 最佳搭子          | ¥16,800  | ~¥13,800 |
| ⭐ 互助之星          | ¥16,800  | ~¥14,800 |
| 有产品/有内容的普通学员 | ¥16,800  | ~¥15,500 |

每一档都已经包含了：
· B产品费用（¥1,299）全额抵扣
· 实操营学费 全额退还
· 各自奖项的额外抵扣

没有隐形消费。没有套路。算得明明白白。"
```

### 第四步：行动召唤（5min）

```

"今天我不在现场卖任何东西。
因为我相信——
这21天的体验，比任何销售话术都有说服力。

如果你感兴趣：

方式一：扫码填一个意向表
告诉我你在想哪条路，我会在48小时内给你回信
（展示二维码）

方式二：直接加我微信1v1聊
有些情况文字说不清楚，我们语音/视频聊
（展示微信二维码）

不管你选哪条路，还是决定先自己跑一阵——
这个群我会留着做**校友录**。
大家随时能找到老同学，互相交流。

但如果想要持续的、有人带着往前走的感觉——
有每月活动、有虾球助手、有1v1诊断——
那是🦐虾玩星球的事。"
```

### 💡 这个环节的关键设计

1. **不逼单，只铺路** — "今天不卖"，降低防备心
2. **数字透明化** — 把每档实付金额亮出来，消除疑虑
3. **双通道行动** — 表单（低压）+ 微信（高压），不同意愿度的学员各有所选
4. **群降级为"校友录"** — 不解散但不主动运营，与虾玩星球差异化（"老同学聚聚" vs "活的持续成长圈子"）
