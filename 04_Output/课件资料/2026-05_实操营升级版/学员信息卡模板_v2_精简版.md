# 📋 学员信息卡模板 · v2 精简版（5月升级）

> **核心升级**：从 v1 的 11 张全手写 → v2 的 **3 张手写 + 8 张 AI 提炼**  
> **设计原则**：人写最重要的（地基卡），其他全部喂料给 AI 提炼  
> **对应**：LLM Wiki 路径 C 分工法则  
> **使用版本**：5月 21 天实操营 ⭐

---

## 一、为什么 v2 要改？

### v1 的问题（旧版 11 张卡都让学员手写）
- ❌ 14 天填不完，学员焦虑
- ❌ 违背 LLM Wiki 范式（"wiki 是 AI 维护的"）
- ❌ 方法论/案例/金句这些本应该 AI 从你过去的文章里提炼

### v2 的设计（路径 C 分工）

```
┌──────────────────────────────────────┐
│   ✋ 手写（仅 3 张地基卡）             │
│   - 一句话定位卡                       │
│   - 故事线卡                          │
│   - 价值观与边界卡                     │
└──────────────────────────────────────┘

┌──────────────────────────────────────┐
│   🤖 AI 提炼（8 张卡，喂料即可）       │
│   - 方法论卡 ← 你的方法论文章         │
│   - 案例卡 ← 你的咨询记录             │
│   - 内容资产卡 ← 公众号历史文章       │
│   - 主题包卡 ← 已发系列内容           │
│   - 用户画像卡 ← 用户对话/反馈        │
│   - 用户问题库卡 ← 咨询录音           │
│   - 决策原则卡 ← 你的语音备忘         │
│   - 产品卡 ← 产品说明书 PDF          │
└──────────────────────────────────────┘
```

**核心金句**：
> "**地基卡你必须手写——AI 替代不了'你成为你'的过程。  
> 装修部分让 AI 干——AI 比你快 100 倍。**"

## ✋ 地基卡 #1：一句话定位卡

**位置**：`wiki/01_我是谁/信息卡/2026-05-XX_定位卡.md`  
**填写时机**：Day 1 直播后立即手写（30 分钟）  
**为什么手写**：定位是"想清楚我是谁"的过程，AI 写出来必空洞

### 📝 模板

```markdown
# 一句话定位卡

## 对象
[你的目标用户：年龄/身份/阶段，越具体越好]

## 场景
[他们处在什么场景下？什么情况会需要你？]

## 问题
[他们遇到什么问题？用他们的原话描述]

## 我是谁（核心一句话）
我是 [你的名字]，致力于帮助 [目标用户]，
通过 [你的方法/产品]，实现 [他们想要的结果]。

## 核心标签
#标签1 #标签2 #标签3 #标签4

## 边界（我不做的事）
- [边界 1]
- [边界 2]
- [边界 3]
```

### 🎯 5 个引导问题（写之前先回答）

```
1. 别人找你，通常是为了解决什么问题？
2. 你最常被朋友如何介绍给别人？
3. 如果只能用 3 个标签定义自己，你选什么？
4. 你最不愿意牺牲的底线是什么？
5. 你希望 AI 说话时的语气关键词是什么？
```

> 💡 **提示**：写完后跑一次 Query 验证：
> ```
> 用我的 wiki/01_我是谁/信息卡/ 给我写一段 100 字的自我介绍。
> ```
> 像你 = 写好了；不像 = 看哪句不对，改 CLAUDE.md 第三节"风格"。

## ✋ 地基卡 #2：故事线卡

**位置**：`wiki/01_我是谁/信息卡/2026-05-XX_故事线卡.md`  
**填写时机**：Day 2-3 安静思考时手写（45-60 分钟）  
**为什么手写**：故事的转折点 + 困境 + 突破，只有你能写

### 📝 模板

```markdown
# 故事线卡（为什么是我）

## 转折点
[你人生/职业的关键转折是什么？发生在什么时候？用一句话概括]

## 当时的困境
[那时候你卡在哪里？最痛苦的是什么？用真实的情绪词描述]

## 我的突破
[你后来是怎么走出来的？用了什么方法？做了什么具体的事？]

## 现在能交付什么
- [核心能力 + 1 个数据]
- [核心能力 + 1 个数据]
- [核心能力 + 1 个数据]

## 这个故事适合吸引谁
1. [类型 1：他们的特征和痛点]
2. [类型 2]
3. [类型 3]
```

### 🎯 4 个引导问题

```
1. 转折点：你人生/职业的关键转折是什么？
2. 当时的困境：那时候你卡在哪里？
3. 我的突破：你后来是怎么走出来的？
4. 现在能交付什么：基于这段经历，你现在能帮别人做什么？
```

> 💡 **提示**：选**真实的、有情绪记忆点的时刻**，不要美化。困境用真实情绪词，允许脆弱。

## ✋ 地基卡 #3：价值观与边界卡

**位置**：`wiki/06_我怎么想/信息卡/2026-05-XX_价值观边界卡.md`  
**填写时机**：Day 6 晚上安静时手写（30-45 分钟）  
**为什么手写**：你的"宁愿少赚也要坚持"的东西，AI 编不出来

### 📝 模板

```markdown
# 价值观与边界卡

## 我坚持的（3 条核心价值观）
1. [价值观 1] - [具体解释：这意味着什么？]
2. [价值观 2] - [具体解释]
3. [价值观 3] - [具体解释]

## 我不做的（边界清单）
1. [边界 1] - [具体说明]
2. [边界 2] - [具体说明]
3. [边界 3] - [具体说明]
4. [边界 4] - [具体说明]

## 我希望 AI 的语气
[3-5 个关键词：温暖 / 真实 / 专业 / 幽默 / 有洞察 / ...]

## ✅ 符合价值观的表达（举 2 例）
- "[正面示例 1]"
- "[正面示例 2]"

## ❌ 违背价值观的表达（举 2 例）
- "[反面示例 1]"
- "[反面示例 2]"
```

### 🎯 3 个引导问题

```
1. 你宁愿慢一点、少赚一点也要坚持的是什么？
2. 你最不能接受 AI 用什么方式表达？你的底线是什么？
3. 如果有人违背你的价值观来找你合作，你会如何拒绝？
```

> 💡 **提示**：反向思考——你最讨厌的营销方式是什么？你最不能接受的合作是什么？这些就是你的边界。

## 🤖 AI 提炼工作流（统一指令）

把素材丢到 `raw/对应类型/` 后，给 Claude Code 这段指令：

```
请扫描 raw/[类型]/ 下的新文件，
按 CLAUDE.md 规则提炼为信息卡，
写到 wiki/[N号房]/信息卡/。

要求：
1. 每张信息卡 ≤ 300 字
2. 含 frontmatter（type/date/source/digest）
3. 标注原始素材来源
4. 同步更新 wiki/index.md（追加索引行）
5. 在 wiki/log.md 记录操作

完成后告诉我：
- 生成了几张卡
- 哪些素材没找到合适的房间（待我手动确认）
```

## 8 张提炼卡的"喂料指南"

### 🤖 #1 方法论卡

**喂什么**：你最拿手的 3-5 个方法论
- 公众号里"X 步法"类型的文章
- 课件 PDF
- 你的方法论笔记

**放到**：`raw/articles/方法论/`

**指令**：
```
扫描 raw/articles/方法论/ 下文件，提炼为方法论卡：
- 每张卡含：方法论名称 / 适用场景 / 核心步骤 / 常见坑 / 适用边界
- 写到 wiki/02_我会什么/信息卡/
```

### 🤖 #2 案例卡

**喂什么**：脱敏后的真实咨询记录
- 学员的咨询逐字稿（去掉真实姓名）
- 你做过的方案 + 结果

**放到**：`raw/咨询逐字稿/`

**指令**：
```
扫描 raw/咨询逐字稿/，提炼为案例卡：
- 每张含：客户背景 / 遇到的问题 / 我的解决方案 / 最终结果 / 关键洞察
- 客户姓名一律脱敏（用化名 A / B / C）
- 写到 wiki/02_我会什么/信息卡/
```

### 🤖 #3 内容资产卡

**喂什么**：公众号历史文章（用 wechat_download 工具一次性入库）

**放到**：`raw/公众号历史文章/`

**指令**：
```
扫描 raw/公众号历史文章/ 下所有文章，按 M1/M2/M3 母题分类，
为每个母题挑出 5-10 篇代表作，写代表作清单到 wiki/03_我说过什么/。
同时抽 30 段最有"我味"的段落 → wiki/03/风格采样/，
抽 20 句标志性金句 → wiki/03/金句库/。
```

### 🤖 #4 主题包卡

**喂什么**：同主题文章 ≥ 3 篇（如"AI 知识库"系列、"产品化"系列）

**放到**：已在 `raw/公众号历史文章/`，按主题打标即可

**指令**：
```
找出 raw/公众号历史文章/ 里同主题 ≥ 3 篇的系列，
为每个系列建一张主题包卡：
- 主题包名称 / 核心问题 / 目标读者 / 代表作 1-3 篇 / 核心观点 / 关键故事
- 写到 wiki/03_我说过什么/主题包/
```

### 🤖 #5 用户画像卡

**喂什么**：
- 学员的报名问卷答复
- 朋友圈/小红书评论区互动
- 私域聊天碎片（脱敏）

**放到**：`raw/chats/用户对话/` + `raw/notes/用户反馈/`

**指令**：
```
扫描 raw/chats/ 和 raw/notes/，提炼为 2-3 类核心用户画像卡：
- 每张含：用户标签 / 基本信息 / 核心痛点 / 他们的原话 / 期待 / 顾虑
- 用户原话用引号 + 出处
- 写到 wiki/05_我面对谁/信息卡/
```

### 🤖 #6 用户问题库卡

**喂什么**：
- 咨询录音转文字
- 社群里被问过最多的问题
- 你的私聊记录

**放到**：`raw/chats/咨询录音/`

**指令**：
```
扫描 raw/chats/咨询录音/，提炼用户高频问题 TOP 10：
- 按"产品咨询 / 技术问题 / 心态困惑 / 决策犹豫"分类
- 每个问题附我的标准回答（如果有）
- 写到 wiki/05_我面对谁/信息卡/
```

### 🤖 #7 决策原则卡

**喂什么**：
- 你的语音备忘录（说出"我为什么这么决定"那种）
- 朋友圈"思考型"动态
- 你做重大决策时的笔记

**放到**：`raw/notes/决策记录/`

**指令**：
```
扫描 raw/notes/决策记录/，提炼为决策原则卡：
- 按 3 大类：内容相关 / 商业相关 / 人生相关
- 每条原则配 1 个正面例子 + 1 个反面例子
- 写到 wiki/06_我怎么想/信息卡/
```

### 🤖 #8 产品卡

**喂什么**：
- 产品说明书 PDF
- 海报文案
- 课程介绍页

**放到**：`raw/articles/产品说明书/`

**指令**：
```
扫描 raw/articles/产品说明书/，每个产品提炼一张产品卡：
- 含：产品名称 / 定位 / 目标用户 / 解决问题 / 核心价值 / 形式 / 价格 / 不适合谁
- 写到 wiki/04_我卖什么/信息卡/
```
