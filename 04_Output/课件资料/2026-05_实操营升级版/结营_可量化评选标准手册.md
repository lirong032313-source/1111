# 🏆 结营颁奖 · 可量化评选标准手册

> **适用**：5月实操营结营仪式（5月30日）
> **奖项**：最佳搭子奖 TOP3 + 互助之星奖 TOP5
> **核心原则**：用数据说话，不凭感觉评

---

## 一、最佳搭子奖（TOP3）

### 1.1 奖项定义

评的是 **"1对1双向陪伴的质量"** —— 不是谁最优秀，而是谁最懂得和另一个人一起变好。

### 1.2 评选维度（满分100分）

| 维度 | 权重 | 数据来源 | 计分方式 |
|---|---|---|---|
| **互评完成度** | **40%** | 互评表/问卷星收集 | 每次按时提交+有实质内容得满分，每缺一次扣5分，满分40分 |
| **搭子互助深度** | **35%** | 互评表中"对方帮助了我什么" | 搭子打分（0-35分）+ 蓉蓉子抽检验证，满分35分 |
| **自身成长可见度** | **25%** | 作业抽查 + 结营展示 | 自己的作业/产出有明显进步、能展示成果，满分25分 |

### 1.3 权益

| 排名 | 权益内容 |
|---|---|
| 🥇 第1名 | 私教升级抵扣 ¥2,000 + 实操营学费全额退还 |
| 🥈 第2名 | 私教升级抵扣 ¥2,000 + 实操营学费全额退还 |
| 🥉 第3名 | 私教升级抵扣 ¥2,000 + 实操营学费全额退还 |

### 1.4 数据采集工具：《搭子互评表》（问卷星）

**发放时间**：结营前3天（5月27日）

#### 问卷题目设计

```
📋 搭子互评表（匿名提交，仅蓉蓉子可见）

=== 基本信息 ===
Q1. 你的名字（用于配对）：
    [单行填空]

Q2. 你的搭子是谁？
    [单选：从学员名单中选择]

=== 核心评分题 ===

Q3. 这21天，你的搭子帮你最多的一件事是什么？
    请具体描述（越具体越好，这直接影响评分）
    [多行填空]

Q4. 你给搭子的帮助中，你觉得最有价值的一次是什么？
    [多行填空]

Q5. 如果满分10分，你给这位搭子打几分？
    [滑块/单选：1-10分]

Q6. （可选）想对搭子说的一句话：
    [多行填空]
```

### 1.5 评分计算方法

```
搭子最终得分 = 互评完成分 + 互助深度分 + 自身成长分

── 互评完成分（40分满）──
✅ 双方都按时提交且内容充实 → 40分
⚠️ 一方未提交或内容敷衍 → 30分
❌ 双方均未提交 → 0分
➖ 每缺一次互评环节记录扣5分

── 互助深度分（35分满）──
基于 Q3+Q4 的回答质量 + Q5 的打分
· 搭子给10分且有具体事例 → 33-35分
· 搭子给8-9分有具体事例 → 28-32分
· 搭子给6-7分 → 20-27分
· 搭子给5分以下 → ≤19分
· 蓉蓉子抽查验证：事例属实不加分不扣分；虚构则 -5~-10分

── 自身成长分（25分满）──
· 结营展示有明显前后对比成果 → 23-25分
· 有可见产出（文章/Skill/工作流截图）→ 18-22分
· 作业基本完成但展示一般 → 12-17分
· 作业完成度低 → 0-11分
```

## 二、互助之星奖（TOP5）

### 2.1 奖项定义

评的是 **"1对多的社群贡献度"** —— 与搭子奖的区别：搭子是深度1对1，互助星是广度1对多。

### 2.2 评选维度（满分100分）

| 维度 | 权重 | 数据来源 | 计分方式 |
|---|---|---|---|
| **社群活跃质量** | **40%** | 蓉蓉子/助教快速过群记录 | 有效发言数（排除"收到""好的"）+ 主动分享经验次数，满分40分 |
| **主动帮助他人** | **35%** | 群记录关键词搜索 | 每帮助1人得5分（搜"帮你看了""试试""可以这样"），上限35分 |
| **正能量氛围贡献** | **25%** | 群记录 + 学员提名 | 鼓励他人/活跃气氛的次数，满分25分 |

### 2.3 权益

| 排名 | 权益内容 |
|---|---|
| ⭐ 第1名 | 私教升级抵扣 ¥1,000 + 实操营学费全退 + 30分钟1v1诊断咨询 |
| ⭐ 第2名 | 私教升级抵扣 ¥1,000 + 实操营学费全额退还 |
| ⭐ 第3名 | 私教升级抵扣 ¥1,000 + 实操营学费全额退还 |
| ⭐ 第4名 | 私教升级抵扣 ¥1,000 + 实操营学费全额退还 |
| ⭐ 第5名 | 私教升级抵扣 ¥1,000 + 实操营学费全额退还 |

### 2.4 数据采集工具：《互助之星提名表》（问卷星）

**发放时间**：结营前2天（5月28日）

#### 问卷题目设计

```
📋 互助之星提名表（可多选+自荐）

=== 基本信息 ===
Q1. 你的名字：
    [单行填空]

=== 提名问题 ===

Q2. 这21天谁在群里帮到你最多？（可多选，最多选3人）
    [多选：从学员名单中选择]

Q3. 谁的发言让你印象最深刻？为什么？
    [填空：姓名 + 原因]

Q4. 如果你要提名一位"互助之星"，你选谁？一句话说明理由。
    [填空]

Q5. （可选）你也可以自荐！如果你觉得自己帮到了很多人，说说你做了什么。
    [多行填空]
```

### 2.5 评分计算方法

```
互助之星最终得分 = 社群活跃分 + 主动帮助分 + 正能量贡献分

── 社群活跃分（40分满）──
有效发言统计（排除水帖如"收到""好的""👍"等）：
· 主动分享经验/方法 ≥8次 → 36-40分
· 有效互动/提问/回应 ≥5次 → 28-35分
· 偶尔参与讨论 2-4次 → 18-27分
· 几乎不发言 → 0-17分

── 主动帮助分（35分满）──
关键词检索群记录（"帮你看了""试试这个""可以这样"
"我查了一下""分享一个""建议你试试"等）：
· 帮助≥7人 → 33-35分
· 帮助5-6人 → 26-32分
· 帮助3-4人 → 18-25分
· 帮助1-2人 → 5-17分
· 未帮助他人 → 0分
*每次帮助需有实质内容，非简单回复

── 正能量贡献分（25分满）──
结合提名票数 + 群内鼓励/活跃气氛行为：
· 被提名≥3次 + 有明显鼓励行为 → 23-25分
· 被提名1-2次 + 有鼓励行为 → 16-22分
· 被提名0次但有鼓励行为 → 10-15分
· 无明显正能量贡献 → 0-9分
```

## 五、注意事项

1. **公平透明**：评选标准提前公开，让所有学员知道怎么评
2. **数据为主**：尽量用量化数据（问卷+群记录），减少主观判断
3. **蓉蓉子最终裁定权**：数据是参考，你有最终决定权
4. **允许兼得**：同一人可以同时获得搭子奖和互助之星奖，但权益取最高档不叠加
5. **颁奖词要个性化**：不要套模板，每个获奖者至少提1个具体事例
6. **提前准备Plan B**：如果某档位分数接近（差距<3分），准备并列或额外名额预案
