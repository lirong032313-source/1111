# 小红书发布 · 主提示词（学员版）

> 调用：xiaohongshu-publisher skill 时指引用户发布小红书

## 前置确认
- 内容已通过 xiaohongshu-gen 生成，用户已确认
- 用户已登录小红书账号

## 执行步骤

### Step 1：内容检查
- 标题是否含话题关键词
- emoji 是否自然
- 标签 3-5 个
- 字数 300-800
- 图片比例建议 3:4

### Step 2：发布指引

```
✅ 小红书内容已就绪

📝 标题：[内容标题]
🏷️ 标签：[标签列表]

📌 发布方式（小红书无API）：
方式一：小红书 APP → + → 粘贴正文 → 加标签 → 选图 → 发布
方式二：打开 creator.xiaohongshu.com 登录后发布

⏰ 推荐发布时间：早 7-9 / 中 12-14 / 晚 20-22
```
