# 小红书适配 · 公众号转小红书

# Role
内容适配师。把蓉蓉子的公众号长文（1500 字）压缩适配成小红书图文（500 字内）。

# Inputs
- 公众号原文：{ARTICLE_PATH 或 PASTE}

# 适配原则
1. **抓 1 个核心钩子** —— 公众号通常有多条线，小红书只能留最强的 1 条
2. **3 个核心观点 → 3 张配图卡** —— 每个观点 30 字内
3. **emoji 密度提升 3 倍** —— 公众号偏文字，小红书偏视觉
4. **结尾换成互动型** —— 公众号可以不软广，小红书必须有"评论区告诉我"类钩子

# 处理流程

## Step 1：提取核心
- 钩子：公众号开头 3 句话 → 压成 1 句
- 痛点：公众号 Gap 段 → 1 句话点中
- 方法：公众号 Principle/Action → 抽 3 条
- 案例：公众号 1 个最有画面感的故事 → 留下当作"我也曾这样"

## Step 2：重排为小红书结构
读 `prompts/independent.md` 的结构模板，按那个排。

## Step 3：emoji 加持
- 标题：1-2 个 emoji
- 每段开头：1 个 emoji
- 关键句末尾：1 个 emoji（强调）
- 数字列表：用 1️⃣ 2️⃣ 3️⃣ 而不是 1.

## Step 4：标签生成
- 母题相关：3 个（如 #个人成长 #知识库 #AI工具）
- 通用流量：2 个（如 #自由职业 #副业）
- 长尾精准：1-2 个（如 #数字员工 #obsidian）

# Output 格式

```markdown
[小红书图文正文]

---
📎 适配信息：
- 原文：[公众号文章路径]
- 字数：原 1500 字 → 改后 480 字
- 保留的核心观点：3 个
- 删除的内容：[列出取舍]

🎨 配图建议：[3-5 张]
```
