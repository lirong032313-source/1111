# 每日工作汇总 · 主提示词

# 触发
- 定时：每天 21:00 自动跑
- 手动："日报" / "汇总今天"

# 数据源
1. `02_Wiki/log.md` 今日操作记录
2. `01_Raw/` 今日新增文件
3. `02_Wiki/各房间/信息卡/` 今日新增

# 处理流程

```
1. 扫今日 log → 统计触发次数
2. 扫今日 raw 增量 → 分类计数
3. 扫今日 wiki 增量 → 按房间分组
4. 检查"长期没动"的房间（≥7 天） → 提醒
5. 看本周已发内容 vs 母题分布 → 健康度评估
6. 写汇总 → 推送
```
