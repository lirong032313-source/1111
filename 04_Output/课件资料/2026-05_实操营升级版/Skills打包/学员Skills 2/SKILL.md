## 使用说明

### 1. 安装 Skills

将本文件夹放入你的 `05_System/skills/` 目录

### 2. 调用方式

在 Claude Code 中输入触发词即可自动调用对应 Skill

### 3. 各 Skill 简介

#### article-gen（公众号草稿生成）
- 触发词：`写公众号`
- 功能：根据选题/素材生成符合蓉蓉子风格的公众号文章草稿
- 包含：写作风格指南、内容结构要求、生成流程

#### wechat-ingest（微信消息入库）
- 触发词：`入库`
- 功能：将微信消息自动识别意图并分类入库到六间房
- 包含：意图识别规则、入库动作流程

#### wechat-publisher（公众号草稿发布）
- 触发词：`发布公众号` / `发到草稿箱`
- 功能：将已生成的公众号文章发布到微信草稿箱
- 注意：草稿箱发布后仍需在微信后台手动发布

#### xiaohongshu-gen（小红书图文生成）
- 触发词：`小红书` / `写小红书`
- 功能：独立生成小红书 或 将公众号文章适配转换为小红书格式

#### xiaohongshu-publisher（小红书草稿发布）
- 触发词：`发布小红书` / `小红书发布`
- 功能：将已生成的小红书内容整理为发布格式，提供发布步骤指引
- 注意：需手动在小红书 APP 中发布

#### daily-brief（每日工作汇总）
- 触发词：`日报`
- 功能：汇总今日任务、消息、待办，生成简洁的工作摘要

#### topic-generator（选题生成）
- 触发词：`选题`
- 功能：基于你的定位和用户痛点，生成选题方向建议

#### content-gen（内容路由器）
- 触发词：（自动判断）
- 功能：根据内容类型自动路由到合适的处理流程
