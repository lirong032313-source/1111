# 选题生成 Skill

> 基于 Wiki 母题库推荐本周选题

## 触发词

- `选题`
- `本周写什么`
- `选题推荐`

## 输入（可选）

用户指定方向：
- 母题（M1/M2/M3）
- 或无指定，全凭推荐

## 处理流程

### 1. 调用 Wiki 底座

读取：
- `02_Wiki/3号房-我说过什么/代表作清单`（参考风格）
- `02_Wiki/3号房-我说过什么/金句库`（引用）
- `01_Raw/` 本周新增素材

### 2. 母题推荐

根据你最近的素材和用户反馈，推荐：

```
📋 本周选题池

1️⃣ 【M1·成长日记】
   "30岁还不知道自己擅长什么"——我也曾是
   来源：木瓜熊咨询碎片 + 1号房故事卡

2️⃣ 【M2·小而美产品】
   从 1v1 咨询里看到的 3 个用户痛点
   来源：5号房用户痛点库

...
```

### 3. 优选推荐

给出 Top 3 选题，每个包含：
- 标题
- 核心观点（1句话）
- 关联的 Wiki 素材
