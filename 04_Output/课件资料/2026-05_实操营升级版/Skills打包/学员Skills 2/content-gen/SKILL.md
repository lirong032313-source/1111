# 内容生成入口 Skill

> 统一的内容生成入口，识别意图并调用对应的生成 Skill

## 触发词

- `写文章`
- `写公众号`
- `小红书`
- `发笔记`
- `转换`
- `公众号转小红书`
- 内容创作相关

## 触发场景

1. **微信端**：通过 ClawBot 收到消息，识别生成需求
2. **网页端**：在 OB 中直接对话生成

---

## 处理流程

### 第一步：意图识别

分析用户消息，识别生成目标：

| 关键词 | 生成目标 |
|--------|---------|
| 公众号、写文章、生成文章 | 公众号草稿 |
| 小红书、笔记、发笔记 | 小红书图文 |
| 公众号转小红书、转换 | 小红书适配 |
| 选题、不知道写什么 | 选题推荐 |

### 第二步：提取素材

从消息中提取：
- 核心主题/选题
- 已有素材（如有）
- 特殊要求（字数/风格/话题）

### 第三步：调用对应 Skill

```
意图识别
   ↓
   ├─ 公众号 → 调用 article-gen
   ├─ 小红书（无原文）→ 调用 xiaohongshu-gen（独立生成模式）
   ├─ 小红书（有原文）→ 调用 xiaohongshu-gen（适配模式）
   └─ 选题 → 调用 topic-generator
```

### 第四步：返回结果

- **微信端**：返回可复制的内容 + 简短说明
- **网页端**：返回 Markdown + 预览

---

## 微信端示例

```
用户发送：写一个关于知识库的小红书

→ 分析意图：生成小红书
→ 提取主题：知识库
→ 调用 Skill：xiaohongshu-gen
→ 返回：

💡 小红书已生成：

# 知识库

[内容...]

#知识库 #个人IP #AI工具
```

---

## 网页端示例

```
用户：帮我写公众号，关于主副业合一

→ 分析意图：生成公众号
→ 提取主题：主副业合一
→ 检查素材：无素材，调用素材库
→ 调用 Skill：article-gen
→ 返回：

---
title: 在轨道中寻找旷野：主副业合一实操指南
---

[公众号正文...]
```

## 输出规范

| 渠道 | 输出格式 |
|------|---------|
| 微信 | 纯文本，可直接复制 |
| 网页 | Markdown + 预览 |

---

## 调用资源

- `article-gen`：公众号草稿生成
- `xiaohongshu-gen`：小红书生成
- `topic-generator`：选题生成
- `02_Wiki/3号房-我说过什么/`（风格素材）
- `02_Wiki/1号房-我是谁/`（身份卡）
- `CLAUDE.md`（写作风格）

---

## 边界规则

- ❌ 不直接发布，只生成草稿
- ❌ 不涉及具体金额/数据
- ✅ 生成后必须经过人工审核确认
- ✅ 调用前先识别意图，避免错误技能调用
