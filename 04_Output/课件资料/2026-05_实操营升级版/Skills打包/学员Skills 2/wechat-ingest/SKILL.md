# 微信消息入库 Skill

> 将微信消息自动识别意图并分类入库

## 触发词

- `入库`
- `[微信收到的消息]`

## 触发场景

当用户在微信中给小龙虾发送消息时，自动调用此 Skill

## 处理流程

### 1. 识别意图

读取消息内容，判断属于哪种类型：

| 类型 | 关键词 | 分类到 |
|------|--------|--------|
| 灵感/想法 | "我想"、"突然想到"、"有个想法" | 01_Raw/灵感碎片 |
| 聊天记录 | 微信对话截图/文字 | 01_Raw/微信聊天碎片 |
| 文章链接 | 转发公众号文章 | 01_Raw/公众号历史文章 |
| 问题/困惑 | "怎么办"、"为什么"、"求解" | 02_Wiki/6号房-我怎么想 |
| 任务/待办 | "待办"、"要完成"、"记得" | 02_Wiki/6号房-我怎么想 |
| 情绪/状态 | "好累"、"焦虑"、"开心" | 02_Wiki/1号房-我是谁 |

### 2. 入库动作

1. 将原始内容保存到对应的 `01_Raw/` 目录
2. 提取 2-5 条核心要点
3. 生成信息卡到对应的 `02_Wiki/` 房间
4. 更新 `index.md`（追加一行）
5. 记录 `log.md`

### 3. 返回确认

向用户返回：
```
✅ 已入库到 [房间]
- 提取了 N 个要点
- 关联到：已有的 X 个页面
```

## 调用资源

- CLAUDE.md（六间房定义）
- index.md（全局索引）
- 对应房间的信息卡目录
