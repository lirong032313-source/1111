## P02 · 三场直播叙事弧线 ⭐ 核心设计

### 设计思路

**不依赖打卡数据统计（可能无法精确统计）**，改用三场直播的内容回顾来串联21天的成长叙事线。每场直播代表一个成长阶段，形成"从0到1"的故事弧线。

### 叙事结构

```
【直播① · 地基搭建】懂我
'AI不是替你干活，AI要先读懂你。'
你们学会了写信息卡，搭建了第一间房。
很多人第一次意识到：原来我的经验是可以被"装起来"的。

【直播② · 养龙虾】养活
CLAUDE.md就是你的龙虾的"喂养手册"。
你开始发现：同样的提示词，有了知识库的AI和没有的是天壤之别。

【直播③ · 数字员工】替我工作
Wiki × Skills × Workflow = 你的数字团队。
有人已经造出了自己的第一个数字员工。
```

### 金句收束

> "21天，从'AI是什么'到'AI怎么为我工作'。这不是三门课，这是一条完整的路。"
>
> "这些不是AI替你们做的。是你们自己做的。我只是给了你们一把铲子，是你们自己挖出了宝藏。"

# 环节02 · 学员成果展示（35min）⭐ 高光时刻

## P04 · 学员邀约说明

选4-5位有代表性、表达能力OK、成果可展示的学员：

| 类型 | 展示内容 | 邀约话术 |
|---|---|---|
| **成长型** | 从完全不会到跑通全流程 | "你的前后对比很有感染力，愿意分享下吗？" |
| **深度型** | 六间房搭建得最完整 | "你的知识库结构特别清晰，想请你做示范" |
| **产出型** | 已有文章/Skill/工作流成品 | "你已经跑通了，是最好的活广告" |
| **互助型** | 搭子互动多、帮助他人 | "大家对你的评价很高，来分享下心得吧？" |

## P05 · 分享框架（统一发给学员）

```
【固定格式 — 提前发给他们】

1. 来的时候我是什么状态？（1-2句）
2. 这21天我最大的1个改变是什么？（核心亮点）
3. 屏幕共享：展示你的成果（2-3min）
   · CLAUDE.md 或等价的配置文件
   · 六间房的结构（哪怕只搭了2-3间也行）
   · 第一篇Output（文章/卡片/任何产物）
4. 一句话送给还没开始/正在犹豫的人
```

## P06 · 点评模板（每人1min）

```
"谢谢XX的分享。我注意到一个细节——
（提炼1个具体的观察点，证明你认真听了）

这恰恰说明了一个道理：
（升华到方法论层面，金句收尾）

给XX鼓掌👏"
```

## P07 · 展示环节执行要点

- **提前彩排**：至少跟每位学员过一遍内容和屏幕共享操作
- **控制节奏**：每人严格5-6min，超时温柔打断："太精彩了，我们待会私下再聊"
- **制造反差**：开场说"来的时候"时强调痛点，展示时强调变化，对比越强烈越打动人
- **录屏留存**：学员同意的话，这些展示素材后续可以做销售案例

## P09 · 🥇 最佳搭子奖 TOP3

### 奖项定义

评的是 **"1对1双向陪伴的质量"** —— 不是谁最优秀，而是谁最懂得和另一个人一起变好。

### 可量化评选标准（满分100分）

| 维度 | 权重 | 数据来源 | 计分方式 |
|---|---|---|---|
| **互评完成度** | **40%** | 互评表/问卷星收集 | 按时提交+有实质内容得满分，每缺一次扣5分，满分40分 |
| **搭子互助深度** | **35%** | 互评表中"对方帮助了我什么" | 搭子打分(0-35)+蓉蓉子抽检验证，满分35分 |
| **自身成长可见度** | **25%** | 作业抽查 + 结营展示 | 有明显进步、能展示成果，满分25分 |

### 权益

| 排名 | 权益 |
|---|---|
| 🥇 第1名 | 私教升级抵扣 ¥2,000 + 实操营学费全额退还 |
| 🥈 第2名 | 私教升级抵扣 ¥2,000 + 实操营学费全额退还 |
| 🥉 第3名 | 私教升级抵扣 ¥2,000 + 实操营学费全额退还 |

### 颁奖流程

1. 宣布："最佳搭子奖——评的是1对1双向陪伴的质量"
2. 公布评选标准（透明化）
3. 从第三名开始揭晓
4. 每位获奖者：上台开摄像头 → 念颁奖词 → 宣布权益 → 感言 → 合影

### 颁奖词模板

```
"第X名 —— @XXX！

这21天里，你做了XX件事让我印象特别深刻：
（具体事例）

你不是一个人在学，你是带着别人一起走。
这就是'搭子'的意义。

🥇 最佳搭子权益：
· 私教升级抵扣 ¥2,000
· 实操营学费 全额退还
恭喜你！"
```

## P11 · 差异化策略：群保留 vs 虾玩星球

> 在颁奖结束后、进入转化环节前，自然过渡到这个话题

**为什么群不解散但不影响虾玩星球转化？**

| 维度 | 实操营群（结营后） | 虾玩星球 |
|---|---|---|
| **定位** | 校友录（老同学偶尔聚聚） | 活的圈子（持续成长） |
| **运营频率** | 偶尔（蓉蓉子一周≤1条链接） | 高频（每月4场活动） |
| **核心价值** | 找到老同学、自由交流 | 直播/工作坊/吐槽大会/1v1诊断 |
| **AI能力** | 无 | 虾球助手24h在线 |
| **价格** | 免费 | ¥1,999/年 |

**关键话术**：
> "群会留着做校友录，大家随时能找到老同学。但如果想要持续的、有人带着往前走的感觉——那是虾玩星球的事。"
>
> **差异化本质**：免费的是"能找到人"，付费的是"有人带着走"。

# 环节05 · 升级路径指引（20min）⭐ 转化关键

## P12 · 两条路的选择

```
"21天结束了。但我知道，在座的很多人——
不是觉得'终于结束了'，
而是觉得'才刚刚开始'。

如果你觉得自己在AI这条路上想继续深耕，
我有两个地方给你：

第一条路：🦐 虾玩星球
这不是课程，也不是训练营。是一个圈子。
每月4场活动，虾球助手24小时在线，
每年还有1次1v1深度诊断。
适合想持续迭代、但暂时不需要深度陪跑的你。

第二条路：💎 私教团
如果你已经尝到了甜头，
而且你有自己的产品或内容基础，
想要更系统地把它变成一套能赚钱的东西——
私教团就是为你准备的。"
```

## P13 · 私教门槛说明

```
但私教团有一个前提门槛——
不是钱的问题，是'有没有东西可以加工'的问题。

我说得很直白：
私教是用来帮你把'已有的'变得更值钱的。
如果你连'已有的'都没有，
那我给你的所有方法都落不了地。

所以私教团的门槛是：
✅ 有产品（课程/咨询/服务/内容，什么都行）
✅ 有内容（文章/案例/经验沉淀，哪怕是雏形）
或者 ✅ 已经购买了我的1v1知识库重构咨询

这三条满足任意一条，我们就可以聊。
```

## P15 · 行动召唤（双通道）

```
"今天我不在现场卖任何东西。
因为我相信——
这21天的体验，比任何销售话术都有说服力。

如果你感兴趣：

方式一：扫码填意向表（低压通道）
告诉我你在想哪条路，48小时内回信
（展示二维码）

方式二：直接加微信1v1聊（高压通道）
有些情况文字说不清楚，语音/视频聊
（展示微信二维码）

不管你选哪条路，还是决定先自己跑一阵——
这个群我会留着做**校友录**。

但如果想要持续有人带着走、
有活动、有圈子、有虾球助手——
那是🦐虾玩星球的事。"
```

### 设计巧思

1. **不逼单，只铺路** — "今天不卖"，降低防备心
2. **数字透明化** — 把每档实付金额亮出来，消除疑虑
3. **双通道行动** — 表单（低压）+ 微信（高压），不同意愿度的学员各有所选
4. **群降级为"校友录"** — 不解散但不主动运营，与虾玩星球差异化
