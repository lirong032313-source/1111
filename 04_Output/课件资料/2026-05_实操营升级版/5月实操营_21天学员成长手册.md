### ⭐ Claude Code 安装详细步骤（备选）

- Mac：按 `⌘ + 空格`，搜索"终端"或"Terminal"，回车打开
- Windows：搜索"PowerShell"打开

#### 第 2 步：粘贴一行命令

在终端里粘贴下面这行，回车：

```bash
curl -fsSL https://claude.ai/install.sh | bash
```

或者如果你装过 Node.js，可以用：

```bash
npm install -g @anthropic-ai/claude-code
```

等待安装完成（大约 1-2 分钟）。

#### 第 3 步：第一次启动

终端里输入：

```bash
claude
```

第一次会跳出登录授权——浏览器会自动打开，扫码或登录 Anthropic 账号即可。

#### 第 4 步：验证安装成功

回到终端，看到类似这样的提示符 = 成功：

```
> 
```

试着输入：

```
你好，自我介绍一下
```

Claude 回应了 = ✅ 安装成功。

#### ⚠️ 常见踩坑

| 问题                               | 解决                           |
| -------------------------------- | ---------------------------- |
| 终端报错 "command not found: claude" | 关闭终端再开一次（PATH 没刷新）           |
| 浏览器登录后回到终端没反应                    | 按 `Ctrl + C` 退出再 `claude` 一次 |
| 国内网络问题                           | 不需要 VPN，但偶尔慢，多试几次            |
| Anthropic 账号没有                   | 用邮箱注册一个，免费额度足够实操营用           |

### 🌐 装 Obsidian Web Clipper（浏览器一键剪藏）⭐

> **作用**：浏览网页时一键把内容保存到 ob-大脑/raw/ —— Day 4-5 喂料给 AI 时这个超快。

#### 第 1 步：装浏览器插件

| 浏览器 | 安装方式 |
|---|---|
| Chrome / Edge | [Chrome 商店搜「Obsidian Web Clipper」](https://chromewebstore.google.com/detail/obsidian-web-clipper/cnjifjpddelmedmihgijeibhnjfabmlf) |

#### 第 2 步：配置保存位置

打开插件设置 → 模板设置：

```
Vault：选你的 ob-大脑
Folder：raw/articles/
```

| 问题 | 解决 |

## 四、🎯 整体工作分工法则（全程都用）⭐

> **这是 21 天最重要的一条原则。** 看完这一节，你才明白这次实操营和别的 AI 课的根本区别。

### 谁做什么？

```
┌──────────────────────────────────────────────┐
│   ✋ 你必须手写（约 3 张地基卡，AI 替代不了）   │
├──────────────────────────────────────────────┤
│  ❶ 01_我是谁/一句话定位.md                    │
│     → 必须自己想清楚"我是谁"                   │
│  ❷ 06_我怎么想/核心价值观.md                  │
│     → 必须从你内心来                          │
│  ❸ CLAUDE.md 第三节"写作风格"                 │
│     → 你和 AI 的契约，必须你写                │
└──────────────────────────────────────────────┘

┌──────────────────────────────────────────────┐
│   🤖 AI 提炼（约 80% 的卡片）                  │
├──────────────────────────────────────────────┤
│  你只管：把素材丢进 raw/                      │
│  AI 自动：提炼 → 写信息卡 → 更新 index.md     │
│                                              │
│  典型场景：                                    │
│  • 02_我会什么 ← 你的方法论文章 / 课件         │
│  • 03_我说过什么 ← 公众号 / 朋友圈历史         │
│  • 04_我卖什么 ← 产品说明书 PDF               │
│  • 05_我面对谁 ← 咨询记录 / 学员反馈           │
│  • 案例库 ← 已发爆款反向沉淀                  │
└──────────────────────────────────────────────┘
```

### 核心金句（请背下来）

> "**地基卡你必须手写——因为这是 AI 替代不了的'你成为你'的过程。**  
> **装修部分让 AI 干——因为这是 AI 比你快 100 倍的事。**"

### 🚨 反例（绝对不要这样做）

❌ **让 AI 替你写"一句话定位"**  
→ AI 写的会非常空洞，因为没有你的真实判断  

❌ **手写所有 80 张信息卡**  
→ 会累死，21 天填不完，违背 LLM Wiki 的核心范式

#### Day 6：手写 06 核心价值观

**任务**：这是地基卡 #2 ——AI 替你写不了。

**晚上找一个安静时间，手写**：

```markdown
# 06_我怎么想/核心价值观.md

## 我做事的 3 条原则
1. ____
2. ____
3. ____

## 我不做什么（边界）
- ____
- ____

## 我跟主流不一样的地方
- 别人都说 ___，但我认为 ___
```

写完后让 AI 跑一次 Query 验证：

```
基于我所有 wiki，给我写一段 200 字的自我介绍。
```

如果 AI 写出来"像你" = ✅ 你的价值观和风格已经被 AI 吸收。

## 九、常见问题 Q&A

### Q1：我没有 Mac，可以参加吗？
A：可以。Windows 也支持 Claude Code 和 Obsidian。指南都通用。

### Q2：我从来没用过命令行，怕不怕？
A：不怕。你只需要会 4 个动作：复制粘贴 / 打开终端 / 输入 `claude` / 输入 `cd 文件夹路径`。Day 1 直播会带做。

### Q3：AI 提炼出来的信息卡不像我，怎么办？
A：3 步走——
1. 看 CLAUDE.md 第三节风格描述够不够具体
2. 在 03/风格采样/ 加更多你的原文
3. 跟 AI 说"这段不像我，因为 [具体原因]，请改成 [新版]"

### Q4：如果我跟不上节奏（错过某天）？
A：不要焦虑。21 天的设计是**关键里程碑**，不是每日打卡——你可以连续休息 2-3 天，下次开工把那几天的素材一起喂给 AI 即可。

### Q5：21 天后我能维护这个系统吗？
A：能。系统是**自维护的**——你只管喂料 + 偶尔调 CLAUDE.md，AI 持续帮你长。

### Q6：跟我的搭子聊不来怎么办？
A：第一周给你们一次"换搭子"的机会（书面申请）。但建议先磨合一下——很多搭子刚开始觉得不合适，跑两天后反而成了最好的搭档。

## 🔟 给你的 3 个开营前心理建设

### 1️⃣ 不要追求"一次写完美"
v1 永远不完美。**跑通 > 完美**。21 天里你会迭代很多版。

### 2️⃣ 不要替 AI 操心
AI 写错的地方就指出来让它改，不要自己默默重写。**这是路径 C 的核心——不是你帮 AI，是你监督 AI 帮你**。

### 3️⃣ 不要孤军奋战
跟搭子说"我卡住了"不丢人。**21 天里，能不能成不取决于你多努力，取决于你被卡住时有没有人接住你**。
