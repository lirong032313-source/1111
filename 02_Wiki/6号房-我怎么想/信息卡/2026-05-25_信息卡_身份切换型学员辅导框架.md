# 信息卡｜身份切换型学员·决策框架

> 归属：6号房 | 来源：六六案例 | 2026-05-25

---

## 三问定位法

遇到身份模糊的学员，问三个问题：

1. **线下你能收到钱吗？** → 能 → 别丢，线上是增量不是替代
2. **你最不想丢的客户是什么样的？** → 描出来 → 以此为05号房起点
3. **你怕什么？** → 怕谈钱/怕上镜/怕失败 → 分场景建智能体攻破

## 产品矩阵原则

```
线下1v1(主力收入) ← 不能动
    ↑
中端工作坊(群体交付) ← 可扩展
    ↑
轻课/音频(被动收入) ← 低门槛试水
    ↑
引流品(免费/49元) ← 筛选客户
```

**核心**：线上产品是漏斗，不是替代品。

## 智能体分场景法

- 内容智能体 → 解决"写不出"（用反向链接机制防车轱辘话）
- 销售智能体 → 解决"不敢谈"（话术+挖需+异议处理）
- 复盘同伴 → 解决"学了就忘"（客户交互→洞察→回流知识库）
