## 一、关于休息 / 暂停 / 说"不"的标准

### 什么时候必须休息（不许 push 自己）

```
身体信号：
  □ 连续 3 天睡眠 < 6 小时
  □ 出现明显躯体化（头痛/失眠/胃痛）
  □ 看到工作消息有"想哭"的冲动
  
心理信号：
  □ 对喜欢的事失去兴趣 ≥ 1 周
  □ 持续无意义感（"我在干嘛"）
  □ 想要"消失一段时间"
  
关系信号：
  □ 跟最亲近的人开始没耐心
  □ 推掉所有社交（不是想独处，是没力气）
```

→ **出现 ≥ 2 条 = 强制休息**，不商量。

### 什么时候可以说"不"

```
场景化判断：
  ✅ 不是我擅长的 → 推荐别人
  ✅ 跟价值观冲突 → 直接拒
  ✅ 让我觉得"被消耗" → 第二次出现就拒
  ✅ 占用时间但回报低 → 评估优先级，多半拒
  ✅ 朋友请求超出我能力 → 真诚说"我帮不了，但 [建议]"
```

### 说"不"的话术库

| 场景 | 话术 |
|---|---|
| 工作合作不接 | "这个我接不下来。如果你需要，[X 朋友] 更合适。" |
| 朋友请求超能力 | "我能帮你 [可以的部分]。[做不到的部分] 我帮不了。" |
| 长辈/家人催 | "我有自己的节奏，等我准备好了一定告诉你。" |
| 学员超期待 | "这个产品做不到 [X]。如果一定要做到，可能需要 [Y]。" |

## 二、关于风险决策（如离职/换赛道）的自我提问清单

### 重大决策前的 7 个问题

```
1. 这件事如果做了，3 年后我会感谢现在的自己吗？
2. 不做的成本是什么？做了的成本是什么？哪个更大？
3. 我现在是因为「想去」还是因为「想逃」？
   → 如果是「想逃」，先解决"逃"的部分再说
4. 我有没有给自己留下退路？（别裸辞）
5. 我的"最坏情况"我能承受吗？
6. 这件事跟我 5 年的方向是同向、垂直、还是相反？
7. 我身边最值得信任的 3 个人怎么看？
```

### 我的"决策金科玉律"

> 🌳 **不鼓励裸辞**。  
> 在轨道中寻找旷野——主副业合一，不脱轨地撒野。
>
> 🌳 **MVP 优先**。  
> 不要等"准备好了"才开始。1 万次准备，都不如 1 次粗糙的开始。
>
> 🌳 **3 个月小步快跑 > 3 年想清楚**。  
> 想 3 年都没动 = 没想 3 年，是没勇气 3 年。
>
> 🌳 **"会不会枯萎" > "会不会增长"**。  
> 一个让你长期枯萎的"机会"，不是机会，是陷阱。

## 三、应对低谷期的自我对话

### 低谷期的判断信号

```
- 持续 ≥ 2 周对喜欢的事提不起兴趣
- 出现"我做这些有什么意义"的反复念头
- 身体明显异常（失眠/食欲变化）
- 跟亲近的人也无法获得力量
```

### 低谷期我会做的 4 件事

1. **承认不假装** —— 不发"积极正能量"朋友圈，告诉最近的 1-2 个人
2. **去自然里** —— 大理那次给我治愈的是松针森林，不是任何方法论
3. **不强迫产出** —— 公众号停更两周比假装"日更"健康得多
4. **回去看 1.0/2.0/3.0 故事** —— 提醒自己"我已经走过来过"

### 我从大理学到的核心

> "**原来我已足够好，我拥有的足够多，叶子和金子同样珍贵。**  
> **原谅自己做错的事错过的人，允许自己软弱和害怕。**"  
> ——金句库 1（个人成长基石）

## 四、与人相处的边界

| 类型 | 我的边界 |
|---|---|
| 工作伙伴 | 价值观一致是必要条件 |
| 客户/学员 | 守边界 + 真诚陪伴，不卑不亢 |
| 家人 | 爱但不绑架（互不绑架） |
| 朋友 | 质量 > 数量，深度连接 1-2 人胜过广撒网 |
| 自己 | **永远站在自己这边**——但允许自己软弱 |
