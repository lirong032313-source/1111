## 🔺 三层交付模型

```
底座搭建(4周)  →  流程固化(4周)  →  内容生产力(4周)
知识库+FAQ助手    新人SOP+训练助手    内容输出+持续迭代
```

## 🔺 行业定制四模块法

| 模块 | 内容 | 产出 |
|---|---|---|
| 01 产品知识库 | 产品介绍/政策/合规 | 知识结构图 |
| 02 客户FAQ库 | 50-100条高频问题 | FAQ清单+标准答案 |
| 03 案例经验库 | 10个标准案例 | 成交逻辑拆解模板 |
| 04 新人SOP库 | 培养路径/话术/指引 | 标准化工作流 |

## 🔺 6步固定会议法

1. Check-in（状态同步）→ 2. 里程碑回顾 → 3. 聚焦议题(只选1个) → 4. 共创落地 → 5. 行动拆解(4个小任务) → 6. 复盘决策

## 🔺 最小行动法则

- 每周2个×30分钟小任务
- 保证最低行动不断线
- 任务在会议后明确拆解
