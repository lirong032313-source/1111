## ¥199 模版交付 SOP（自学型）

```
1. 用户付款
2. 邀请进入飞书空间 + 发模版拷贝链接
3. 附《使用说明文档》指导一步步填
4. 后续模版更新 → 社群/邮件通知
5. 不含 1v1 咨询，可升级到产品咨询
```

**核心承诺**：1-2 周内拥有可用初始版本 + 持续迭代的骨架

## ¥999 咨询交付 SOP（1v1 90 分钟）

```
会前（资料收集）：
  用户提供：
  - 简要背景自述
  - 目前在做/做过的产品/服务
  - 已有的知识库截图/结构（如果有）
  我预先阅读并做初步标注

通话中（共创梳理）90 分钟：
  Part 1（30 min）：收拢"你到底有哪些可卖的能力和资产"
  Part 2（30 min）：一起绘制【个人产品线地图】草稿
  Part 3（30 min）：敲定一个"2-4 周内可启动的最小产品实验"

通话后（整理回传）：
  飞书整理：
  -《知识库 × 产品线梳理记录》
  -《产品线地图初稿》（文字版 + 简图）
  发给用户作为接下来行动的参考底稿
```

**核心承诺**：带走的不是几句建议，是可落地执行的产品线初稿

## ¥1899 实操营交付 SOP（21 天 · 详见课件）

```
开营前 7 天：
  - 问卷调研（已有 12 题精简版）
  - 搭子配对（强弱混搭）
  - 找大象任务清单
  - 14 天成长手册

开营中 21 天：
  - Phase 1 (Day 1-7)：搭知识库底座 + 直播 1
  - Phase 2 (Day 8-14)：领养小龙虾 + 直播 2
  - Phase 3 (Day 15-21)：内容工厂 + 直播 3
  - 每日打卡 + 关键里程碑作业 + 搭子互助

结营当天：
  - 每人展示成果（龙虾 + Wiki + 已发内容）
  - 颁发证书 + 优秀学员特别奖
  - 7 份结营礼物（Skills 包/CLAUDE.md 模板库/排错手册等）
  - alumni 社群入驻
```

## ¥16800 私教交付 SOP（12 周）

```
形式组合：
  - 每 1-2 周固定 1v1 深度会议
  - 飞书共创空间（共享知识库 + 看板 + 任务）
  - 项目期间不限量文字交流（设节奏边界）
  - 阶段性小里程碑

里程碑节奏：
  第 3 周：完成知识库重构（v2.0）
  第 6 周：智能体初版上线
  第 10 周：1 个数字产品跑完一轮（含调研→设计→交付→复盘）

典型交付物：
  ✅ 飞书知识库 v2.0（6 间房 + 核心内容整理好）
  ✅ 1-N 个专属智能体（商业向/产品向/内容向/人生向）
  ✅ 数字产品项目档案（设计说明书 + 知识库映射 + AI 提示词集 + 复盘）
```

**核心承诺**：完成一次"硬件+软件"升级——硬件是知识库+智能体系统，软件是你独立处理 AI 的能力

## 共同的"我会做 / 我不会做"

| 我会做 | 我不会做 |
|---|---|
| 提供清晰的下一步 | 替你做决策（决策权永远在你） |
| 给你结构化的方法 | 替你执行所有任务 |
| 用真实经验帮你少走弯路 | 承诺"一定大卖" |
| 在边界内深度陪伴 | 24小时随叫随到 |
| 整理 + 回传文字记录 | 写完后让你照搬不修改 |
