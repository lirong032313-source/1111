# 信息卡｜私教陪跑 · 行业定制版交付示例

> 类型：产品案例卡 · 来源：华总(保险) · 归属：4号房
> AI调用场景：向潜在客户展示陪跑交付形态 / 产品介绍素材

---

## 案例速览

| 维度 | 内容 |
|---|---|
| **学员** | 华总，香港保险团队leader |
| **周期** | 12周 + 2月迭代期 |
| **交付物** | 知识库四模块 + 3大智能体 + 新人SOP |
| **节奏** | 双周1v1会议(6次) + 每周2个小任务 |
| **协作平台** | 飞书空间共创 + 微信紧急沟通 |

## 交付形态一览

| 模块 | 具体产出 |
|---|---|
| 📚 知识库 | 产品知识/FAQ(50-100条)/案例(10个)/SOP |
| 🤖 智能体 | FAQ助手(P0)→训练助手(P1)→内容助手(P2) |
| 📋 SOP | 新人培养路径(考试→入职→课程→实战) |

## 适用同类场景

- 保险/金融/教育/咨询等 知识密集型行业
- 有团队管理需求（新人培养、标准化输出）
- 需要AI赋能但不知从何入手
