### 【A】14天 六间房知识库AI实操营 — 引流课

#### 基本信息
| 项目 | 详情 |
|---|---|
| **主标题** | 14天 六间房知识库AI实操营 |
| **副标题** | 把散落的经验，搭成AI能读懂、你能复用的「数字大脑」 |
| **价值主张** | 让AI真正成为你的「第二大脑」，而不是聊天玩具 |

#### 目标用户痛点（你是不是也卡在这里？）
- ❌ 内容越积越多，但找不到、用不上
- ❌ AI输出无法用，不像你的风格（提示词写了一堆还是瞎编）
- ❌ 想持续做自媒体，但每次都从零开始
- ❌ 有经验有案例，但不知道怎么沉淀成产品
- ❌ AI工具用了一堆，但没有真正提升生产力

#### 产品核心：六间房框架
> 一间房 = 一个关键认知与能力模块；六间房 = 构成AI理解你、像你思考、替你辅助工作的数字大脑

| 房间 | 内容 | AI调用效果 |
|---|---|---|
| 第1间房：个人底层档案 | 你是谁、做什么、擅长什么、价值主张 | AI输出时自动带上身份标签和定位 |
| 第2间房：专业能力库 | 方法论、框架、工具、流程 | AI能调用专业知识，不瞎编 |
| 第3间房：内容资产库 | 文章、案例、金句、素材卡片 | AI引用真实内容，保持风格一致 |
| 第4间房：产品服务体系 | 产品结构、交付流程、客户反馈 | AI帮你做产品设计和客户沟通 |
| 第5间房：用户市场洞察 | 目标用户画像、痛点、需求、场景 | AI输出精准对准用户需求 |
| 第6间房：决策风格模型 | 思考方式、判断标准、价值观 | AI建议更符合决策逻辑 |

#### 14天实操路径
```
第1-3天：地基搭建 → 六间房初始化 + 个人档案梳理 + 信息卡写法训练
第4-7天：内容填充 → 专业能力库 + 内容资产库 + 产品服务体系
第8-10天：AI调教 → 知识库对接AI + 提示词工具箱 + 风格测试优化
第11-14天：工作流跑通 → 半自动供稿流程 + 创作实战 + 系统迭代
```

## 📋 5. 服务流程 SOP

### 咨询流程（1v1 产品设计）

```
Step 1: 需求初筛（15min 免费）
   ↓ 了解基本情况，判断是否匹配
   
Step 2: 深度诊断（60min 付费）
   ↓ 拆解用户现状/资源/卡点/目标
   
Step 3: 方案设计（24-48h 内交付）
   ↓ 输出个性化产品方案（定位/定价/MVP/SOP）
   
Step 4: 方案对齐（30min）
   ↓ 讲解方案，答疑，确认方向
   
Step 5: 跟进支持（2周内）
   ↓ 解答落地过程中的问题，确保执行
```

### 实操营流程（14天）

```
Day 0: 开营仪式 → 模板分发 → 任务说明
Day 1-14: 每日任务打卡 + 社群答疑
Week 1: 直播共创①（90min）— 地基搭建演示
Week 2: Office Hour①（60min）— 问题集中解答  
Week 2: 直播共创②（90min）— AI调教实战
Week 2: Office Hour②（60min）— 个性化辅导
Day 14: 结营验收 → 证书颁发 → 升级通道开启
```

### 私教陪跑流程（3个月）

```
Phase 1 筛选: 申请表 → 面谈(30min) → 审核通过 → 签约付款
Phase 2 启动: 六间房现状评估 → 目标对齐 → 3个月路线图制定
Phase 3 执行: 每周1v1(60min) + 期间异步支持 + 里程碑验收
Phase 4 收尾: 成果交付 → 迭代建议 → 长期维护方案
```

### 决策流程（是否接某个客户/项目）

```
1. 价值观匹配吗？（不割韭菜/真实开放）
2. 我能真正帮到他吗？（不是所有需求都接）
3. 时间允许吗？（主业+副业的精力分配）
4. 价格合理吗？（不低于底线价）
5. 有没有长期价值？（是否可能成为案例/合伙人）
```
