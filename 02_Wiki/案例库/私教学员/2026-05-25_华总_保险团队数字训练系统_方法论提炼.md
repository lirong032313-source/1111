## 蓉蓉子交付框架（可复用）

### 🔺 三层交付模型

```
12周陪跑 = 底座搭建(4周) + 流程固化(4周) + 内容生产力(4周)
           └ 知识库+FAQ助手    └ 新人SOP+训练助手   └ 内容输出+持续迭代
```

### 🔺 双系统架构（行业定制版）

| 系统 | 角色 | 内容 |
|---|---|---|
| **飞书** | 底座（净资产） | 产品知识/FAQ/案例/SOP 四大模块 |
| **YouMind** | 对话检索（应用层） | 新人快速查询、智能问答 |

### 🔺 固定会议结构（6步法）

1. Check-in（状态同步）
2. 里程碑回顾（任务完成度）
3. 本次聚焦议题（只选1个）
4. 共创与落地（产出可用物）
5. 行动拆解（下两周4个小任务）
6. 复盘与边界（决策记录）

## 可复用方法论

### 1. 行业定制四模块法
```
01 产品知识库 → 产品介绍/政策/合规
02 客户FAQ库   → 50-100条高频问题
03 案例经验库   → 10个标准案例+成交逻辑
04 新人SOP库   → 培养路径/话术/指引
```

### 2. 最小行动法则
- 每周2个30分钟小任务
- 保证"最低行动"不断线
- 任务在每次会议后明确拆解

### 3. 智能体三场景优先级
- P0: FAQ助手（降新人门槛）
- P1: 训练助手（产品问答+模拟）
- P2: 内容助手（脚本+案例拆解）
