# 蓉蓉子 · 品牌视觉规范 v2.0

> **定位**：AI 一人公司分身架构师
> **制定日期**：2026.05.18

---

## 一、品牌视觉核心概念

**一句话定调**：把你的经验，变成另一个你。

你卖的不是"AI 工具课"，是"AI 分身"。所以整个视觉的核心隐喻只有一个——**镜像**。

你的经验被复制、结构化，最后变成另一个替你工作的数字生命体。整套视觉都要传递这种"镜像感"和"结构感"。

### 视觉关键词（设计师必看）

- 镜像 / 分身 / 对称
- 六边形结构 / 蜂巢 / 容器
- 冷静的科技感 + 温热的生命感
- 可被"读取"的数据质感

## 二、品牌配色方案

### 色彩系统

| 角色 | 色名 | 色号 | 使用场景 |
|------|------|------|---------|
| **主色** | 深海黑蓝 | `#0F172A` | 背景、主标题、品牌大色块。比上一版的蓝更深，代表"一人公司的地基"——扎实、冷静、不轻浮。 |
| **辅色** | 电光蓝 | `#3B82F6` | 按钮、链接、关键数据、高亮文字。传递 AI 的技术感和速度感。 |
| **强调色** | 琥珀金 | `#F59E0B` | 价格标签、行动指令按钮、核心卖点小标记。代表"你的经验变成金子的那个瞬间"。 |
| **中性色** | 霜白 | `#F8FAFC` | 卡片背景、大面积留白。 |
| **中性色** | 炭灰 | `#1E293B` | 正文文字、次要信息。 |

### 使用比例

> 深海黑蓝 40% → 霜白 30% → 炭灰 20% → 电光蓝 7% → 琥珀金 3%

**琥珀金是奢侈品逻辑，少即是贵。** 只在"价格"和"最核心的那个卖点"上出现，用一次就是一次视觉锤。

## 三、品牌 Logo 设计方向（3 个方案，按推荐顺序）

### 方案 1：镜像脑 ❤️ 最推荐

**概念**：左右两个完全对称的侧脸轮廓，共用一个大脑区域。左半边是实线的"人"，右半边是像素点阵构成的"分身"。中间的脑区用六边形网格填充，微微发光。

**寓意**：你就是它，它就是你。一人公司分身具象化。

**字体**：Logo 下方配"蓉蓉子"三个字，用无衬线几何体（如 HarmonyOS Sans Bold），字间距拉宽。

**适用**：主 Logo，放公众号头像、海报厂牌。

---

### 方案 2：六间房容器

**概念**：六个错落叠放的六边形，其中一个微微打开，有数据流从中溢出（小光点从六边形飘出来）。

**寓意**：六间房是你的知识容器，里面的东西在自动流动、自动工作。

**字体**：Logo 右侧配竖排"六间房"签名体。

**适用**：知识库封面、社群图标、产品水印。

---

### 方案 3：虾球签名标

**概念**：一只极简线条的小虾，虾尾卷成一个"R"形，虾眼是一颗发光的六边形。

**寓意**：虾球是你的数字分身，可爱但聪明。

**字体**：手写感签名体"Rong"。

**适用**：AI 虾球助手的头像、表情包、周边贴纸。

## 四、字体规范

| 场景 | 字体 | 字重 | 说明 |
|------|------|------|------|
| 海报主标题 | 思源黑体 / HarmonyOS Sans | Bold | 权威、干净、有分量 |
| 海报副标题 | 思源黑体 | Regular | 和主标题形成反差 |
| 正文 | 苹方（中文） / Inter（英文） | Regular | 手机阅读最友好 |
| 定价数字 | Inter | Extra Bold | 价格要让人一眼咬住 |
| 金句引用 | 站酷文艺体（备选） | 手写感 | 只在朋友圈长文配图用，不要在海报里出现 |

## 五、品牌视觉锤

### 视觉锤 1：六边形光点 ✦

一个微小的 ✦ 形状，琥珀金色，带一点弥散光晕。

**使用规则**：所有海报、PDF、朋友圈配图中，核心卖点文案末尾，都加这个光点。

**目的**：三个月后，任何人看到一个琥珀金色的六边形光点，就想到蓉蓉子。这是你的视觉钉子。

---

### 视觉锤 2：虾球的六边形眼镜

任何场景下出现"AI 虾球助手"，它的形象永远戴着一副六边形框架的眼镜。

**目的**：六边形结构+AI 分身的双重绑定。可爱，但不可复制。

## 六、海报版式规范

### 构图原则：左右分屏

```
┌─────────────────┬─────────────────────────────┐
│                 │                             │
│   深海黑蓝底     │         霜白 / 图表 / 照片   │
│                 │                             │
│   主标题（白）   │                             │
│   副标题（蓝）   │                             │
│   Logo          │                             │
│                 │                             │
│                 │                             │
└─────────────────┴─────────────────────────────┘
     40%                   60%
     深海黑蓝               霜白/内容
```

- **分割线**：一条极细的电光蓝竖线，中间一颗琥珀金六边形光点
- **背景肌理**：在所有深海黑蓝的色块上，叠加一层半透明的六边形网格纹路（透明度 5%-10%）。不能太明显，但要有。让人感觉"这下面有结构"。

## 七、应用场景速查表

| 场景 | 尺寸 | Logo 版本 | 配色 | 备注 |
|------|------|----------|------|------|
| 公众号头像 | 1:1 方形 | 方案 1 · 镜像脑 | 深海黑蓝底 | 只放 Logo，不放字 |
| 视频号头像 | 1:1 方形 | 方案 3 · 虾球 | 深海黑蓝底 | 亲民，增加辨识度 |
| 知识库封面 | 16:9 横图 | 方案 2 · 六间房 | 深海蓝+琥珀金 | 加产品名称 |

| 朋友圈配图 | 1:1 或 3:4 | 虾球六边形眼镜 | 自定义 | 金句+光点收尾 |
| 直播背景 | 16:9 横图 | 海报主视觉 | 深海黑蓝 | 简洁，不加字 |
| 微信客服头像 | 1:1 方形 | 方案 3 · 虾球 | 琥珀金色底 | 要和你的主号区隔 |

## 八、避雷指南（设计师不能犯的错）

- ❌ **不要用纯白背景** → 用霜白 `#F8FAFC`。纯白刺眼，显得廉价。
- ❌ **不要用渐变色做 Logo** → Logo 一定用纯色，深海黑蓝或纯白反白。
- ❌ **不要在海报里放超过 3 种字体** → 主标题、副标题、正文各一种，够了。
- ❌ **不要用卡通字体** → 你不是卖儿童编程课的。
- ❌ **不要用 Unsplash 上那种"微笑办公女郎"配图** → 要用图就用你自己的照片，或者纯文字+结构图。没有合适的照片，宁可不出镜。
