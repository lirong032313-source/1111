## 🚀 使用方式

### 方式一：Web 对话界面（面向客户）

直接在浏览器打开 `chat.html`，或托管到静态服务后分享链接。

```
file:///Users/rong/Desktop/ob大脑/05_System/AI分身/chat.html
```

**特点**：
- 预置欢迎卡片 + 快捷提问按钮
- 知识检索引擎——自动匹配六间房内容
- 产品推荐卡片
- 金句随机注入
- 移动端适配

### 方式二：WorkBuddy Skill（石榴内部使用）

Skill 已安装在 `~/.workbuddy/skills/avatar-chat/`。对话中说以下任意一句即可激活：

```
「切换分身模式」
「用蓉蓉子的身份回答」
「假设你是蓉蓉子，怎么回复这个客户」
```

Skill 会自动加载 AVATAR.md + knowledge-index.json + 读取对应 Wiki 文件。

### 方式三：手动激活（复制粘贴）

将 AVATAR.md 的内容作为系统提示词注入到任何对话型 AI 中（ChatGPT/Claude 等）。
