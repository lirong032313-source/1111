# SOP：5步内容工作流 · 小蜜版

> **角色**：小蜜（📝 内容分身）负责执行
> **适用**：公众号文章 / 小红书笔记 从碎片到分发
> **核心**：5步法 + 2个确认点 + 1个沉淀动作
> **教学场景**：学员看完就能套用到自己的数字员工

---

## 总览

```
微信发碎片 / 朋友圈素材
        ↓
  ① 小蜜出选题
        ↓ ⏸️ 确认点1：你选哪个？
  ② 小蜜出大纲
        ↓
  ③ 小蜜出初稿 → 自动存 OB 草稿箱
        ↓ ⏸️ 确认点2：初稿OK？发哪里？
  ④ 你改20%（语气/故事/金句）
        ↓
  ⑤ 小蜜分发 → 公众号草稿箱 / 小红书
        ↓
  好内容反哺 Wiki
```

---

## 角色分工

| 谁 | 干什么 | 不干什么 |
|---|---|---|
| **你（CEO）** | 给碎片、选选题、改20%、确认分发 | 不从头写 |
| **小蜜（内容分身）** | 出选题、出大纲、出初稿、存草稿、发平台 | 不替你做决策 |

> **教学要点**：小蜜不是"AI工具"，是你的内容分身。她调你的Wiki（3号房风格/1号房身份/5号房用户痛点），所以写出来像你。

---

## Step 1 → 给素材

**你做的事**：把碎片/灵感/朋友圈发给小蜜。

```
示例：
"最近朋友圈发了一条，说六扇门=一人公司的军火库，
有了底座，工具是选择。帮我写一篇公众号。"
```

**小蜜做的事**：读取碎片 → 调 Wiki 母题（M1/M2/M3）→ 匹配本周素材。

---

## Step 2 → 出选题 ⏸️

**小蜜输出**：

```
📋 选题池（基于你的碎片 + Wiki母题）

1️⃣ 【M3·AI智囊团】
   "六扇门=一人公司的军火库：有底座，工具是选择"

2️⃣ 【M2·小而美产品】
   从"攒工具"到"搭军火库"：一人公司最值得投的一笔时间

3️⃣ 【M1·成长日记】
   我花8个月搭六间房，就是为了今天能说"工具你随便换"

4️⃣ 【M1×M3】
   一人公司不需要五个人，需要一座军火库里的五个数字分身

5️⃣ 【M2】
   为什么越专业的人越容易掉进"工具坑"？因为没底座
```

### ⏸️ 确认点 1：你选哪个选题？

> **学员操作**：选 1 个。也可以说"1和3合并"或"换个方向"。

---

## Step 3 → 大纲 + 初稿

**小蜜做的事**：用 GAP 框架出大纲 → 调 03 风格 + 01 身份 → 生成初稿 → **自动存 OB 草稿箱**。

### GAP 大纲

| 框架 | 说明 |
|---|---|
| **Gap（问题）** | 用户不停换AI工具，越换越焦虑 |
| **Analysis（拆解）** | 误区：以为是工具不够好 → 真相：工具不认识你 |
| **Principle（原则）** | 底座 > 工具；数字分身从知识库里长出来 |
| **Action（行动）** | 先搭一个房间，10篇文章就能感受到 |

### 初稿存储规则

```
OB路径：04_Output/自媒体内容/公众号/草稿箱/
文件名：YYYY-MM-DD_公众号_{标题关键词}-小蜜初稿.md
```

### 初稿结构

```
① 开头钩子（100-200字）：朋友圈引出 → 核心观点
② 正文（3-4段，每段300-500字）：
   - Gap：你没缺工具
   - Analysis：军火库不是堆出来的
   - Principle：从一间房到五个人
   - Action：从3号房开始
③ 结尾升华：金句 + 个人介绍 + 产品钩子
```

## Step 4 → 你改20% ⏸️

### 润色检查表（只改这4个地方）

| 检查 | 问题 | 怎么改 |
|---|---|---|
| ☐ 语气像我吗？ | 读出来听听 | 加 1-2 句你的口头禅 |
| ☐ 专业准确吗？ | 有没有外行话 | 改成你的表达方式 |
| ☐ 用户视角吗？ | 能不能戳中读者 | 加 1 个真实学员故事 |
| ☐ 价值观一致吗？ | 有没有越线 | 删掉"正确的废话" |

> **教学要点**：不是从头改到尾。AI 写 80%，你改 20%。这 20% 才是"为什么是你"。

### ⏸️ 确认点 2：初稿 OK？发哪个平台？

| 平台 | 格式要求 |
|---|---|
| 公众号 | 保留原文风格，完整正文 → 发到草稿箱 |
| 小红书 | emoji 密、节奏快、第一句钩子、带标签 |

## Step 5 → 小蜜分发

### 公众号草稿箱

```
小蜜 → 调微信官方API → 上传封面 → draft/add → 草稿箱
```

> 你在 mp.weixin.qq.com 后台预览并手动发布。

### 小红书

```
小蜜 → 生成小红书图文版 → 你复制粘贴到小红书App
```

### Wiki 反哺

```
文章发布后：
├─ 核心方法论 → 2号房-方法论卡
├─ 金句 → 3号房-金句库
├─ 新案例 → 案例库
└─ 选题 → 选题池
```

## 常见问题

| Q | A |
|---|---|
| 选题都不满意怎么办？ | 说"换一批"或"往M2方向再出3个" |
| 初稿太AI怎么办？ | 不是删了重写，是加你的口头禅和真实故事 |
| 没搭Wiki怎么办？ | 先搭3号房，10篇文章就够了 |
| 没公众号怎么办？ | 不用强求。目标是"跑通能力"，能跑通未来想发再发 |
