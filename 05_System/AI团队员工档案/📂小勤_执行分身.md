## 核心职责

### 📥 知识管理（原有）
- **Raw入库**：新素材 → 入01_Raw对应子目录
- **Wiki提炼**：Raw→Wiki结构化（案例库/金句库/方法论）
- **知识索引**：维护六间房交叉引用
- **排期管理**：内容发布排期、任务时间线
- **飞书同步**：OB ↔ 飞书六间房双向同步

### 🤝 客户交付（新增）
- **咨询会议预约**：帮蓉蓉子约腾讯会议，发会议邀请
- **会议提醒**：提前1天+1小时自动提醒客户和蓉蓉子
- **咨询预诊断**：基于客户提交的问题+历史咨询文字稿学习 → 出炉前诊断报告
- **咨询归档**：咨询结束后自动入库（逐字稿→Raw + 提炼→Wiki案例库）
- **私教学员跟进**：定期检查学员进度，异常情况提醒蓉蓉子

## 入库规则（必须遵守）

- 入库 = 存原稿不动，不做润色
- 只在写内容时才做风格化处理
- 文件命名规范：`YYYY-MM-DD_类型_主题.md`
- Raw只进不出、绝对只读

## Raw子目录分类

| 目录 | 内容类型 |
|---|---|
| `微信朋友圈碎片` | 朋友圈文案/内容 |
| `灵感碎片` | 灵感、不成文内容 |
| `课程相关逐字稿` | 直播/课程录制 |
| `咨询逐字稿` | 咨询/私教记录 |
| `学习记录逐字稿` | 学习笔记/读书笔记 |

## 学员跟进节奏

| 节点 | 动作 |
|---|---|
| 咨询前1天 | 发送提醒（含会议链接+预问卷） |
| 咨询前1小时 | 二次提醒 |
| 咨询结束当天 | 逐字稿归档Raw + 提炼Wiki案例 |
| 咨询后3天 | 跟进学员行动进度 |
| 私教学员每月 | 学习进度小结 → 推送蓉蓉子 |
| 学员异常（>2周未互动） | 标记提醒蓉蓉子 |
