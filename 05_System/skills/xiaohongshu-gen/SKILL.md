# 小红书图文生成 Skill

> 将公众号内容适配转换为小红书图文，或独立生成小红书内容

## 触发词

- `小红书`
- `图文适配`
- `发小红书`
- `笔记`

## 触发场景

1. **独立生成**：直接生成小红书（用户提供主题/素材）
2. **适配转换**：将公众号文章转换为小红书格式

## 输入要求

用户提供：
| 场景 | 必填 | 可选 |
|------|-----|------|
| 独立生成 | 主题/选题 | 关键点素材 |
| 适配转换 | 公众号原文 | 是否需要配图 |

## 处理流程

### 模式 A：独立生成

```
1. 提取主题 → 调用 Wiki 找素材 → 生成初稿 → emoji 加持 → 输出
```

```
1. 提取主题/选题
2. 调用 Wiki 底座：
   - 风格采样
   - 金句库
   - 身份卡
3. 使用小红书结构生成
4. emoji 加持
5. 格式调整
```

### 模式 B：适配转换

```
1. 核心提取 → 3个观点 → emoji加持有 → 输出
```

### 1. 核心提取

从公众号文章中提取：
- 开头 3 句话的钩子
- 3 个核心观点（每句 30 字内）
- 1 个行动建议

### 2. emoji 加持

增加 emoji 密度：
- 💡 观点提示
- 👉 行动指引
- ✨ 强调
- 📌 标记

### 3. 格式转换

小红书格式要求：
- 标题：# 话题 + 关键词
- 正文：分段 + emoji
- 结尾：# 话题标签

### 4. 配图建议

如果用户需要配图，给出：
- 需要的截图类型
- 推荐的图片风格

## 输出格式

```markdown
# [话题] 关键词

[钩子 3 句话]

👇 [核心观点 1]

[核心观点 2]

[核心观点 3]

#标签1 #标签2 #标签3
```
