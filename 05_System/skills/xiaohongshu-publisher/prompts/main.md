# 小红书发布 · 主提示词

> 调用：xiaohongshu-publisher skill 时准备小红书内容并指引发布

## 前置确认
- 用户已确认内容无误
- 内容来自 xiaohongshu-gen skill 产出

## 执行步骤

### Step 1：内容格式检查
检查是否符合小红书发布规范：
- 标题格式：建议含话题关键词
- emoji 使用是否自然
- 标签数量：3-5 个
- 正文字数：300-800 字
- 如有图片，检查比例建议 3:4

### Step 2：配图建议（可选）
如用户需要配图，推荐：
- 通义万相：https://xianghe.aliyun.com
- 内容工厂配图功能

### Step 3：生成发布指引

```
✅ 小红书内容已准备就绪

📝 标题：[内容标题]
📝 正文：[正文预览]
🏷️ 标签：[标签列表]

📌 发布方式（小红书无API，需手动或浏览器发布）：

方式一：手动发布
1. 打开小红书 APP
2. 点击「+」发布
3. 粘贴正文
4. 添加话题标签
5. 选择配图
6. 点击发布

方式二：电脑浏览器（如有 xiaohongshu-ops skill）
使用浏览器自动化登录小红书创作者中心发布

⏰ 推荐发布时间：早 7-9 / 中 12-14 / 晚 20-22
```
