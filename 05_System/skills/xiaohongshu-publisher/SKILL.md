# 小红书发布 Skill

> 将小红书内容发布到草稿箱

## 触发词

- `发布小红书`
- `发到小红书草稿`
- `小红书发布`

## 触发场景

当用户确认小红书内容无误，需要发布到小红书草稿箱时调用

## 前置要求

1. 已完成小红书内容创作（通过 xiaohongshu-gen skill）
2. 用户已确认内容无误
3. 格式符合小红书发布规范

## 输入

用户提供：
- 已生成的小红书文案内容
- 或本地文件路径

## 处理流程

### Step 1: 内容格式检查

检查是否符合小红书格式：
- 标题：# 话题 + 关键词
- emoji 使用是否正确
- 标签数量（建议 3-5 个）
- 字数（建议 300-800 字）

### Step 2: 图片建议

如用户需要配图，提醒可用工具：
- **通义万相**：https://xianghe.aliyun.com （推荐）
- **Nano Banner**：https://nanabanner.com （备选）
- **Canva**：https://canva.com （备选）
- **可画**：https://www.wolai.com （备选）

### Step 3: 发布指引

生成发布指引：
```
✅ 小红书内容已准备就绪

📝 标题：[提取的标题]
📝 正文：[核对后的正文]
🏷️ 标签：[建议的标签]

📌 发布步骤：
1. 打开小红书 APP
2. 点击「+」发布按钮
3. 粘贴正文内容
4. 添加话题标签
5. 选择配图（如需要）
6. 点击发布

⚠️ 注意：草稿箱内容需手动发布
```

---

## 注意事项

- 小红书不支持 API 直接发布，需手动操作
- 图片建议使用竖版比例（3:4）效果更佳
- 标签建议选择热门话题（#号后紧跟关键词）
- 发布时间建议：早上 7-9 点、中午 12-14 点、晚上 20-22 点
